# Synthetic Sales Conversation Fixtures

These files are fake sales conversations for testing Collection upload, parsing, indexing, and cited retrieval.

Suggested Collection ID:

```text
enterprise-sales-conversations
```

How to test:

1. Sign in as `editor` or `admin`.
2. Open Collections.
3. Create a Collection with ID `enterprise-sales-conversations`.
4. Upload the transcript files in this folder.
5. Open Tool Test Bench.
6. Request context type `audience-profile`, `value-proposition`, or `brand-messaging` with a query such as:
   - `buyers worried about stale AI context`
   - `approved truth versus raw transcripts`
   - `enterprise governance objections`
   - `semantic search with cited passages`

All company names, people, quotes, and details are synthetic.
