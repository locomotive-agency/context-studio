# AI Context Governance Launch Goals

The launch campaign introduces Context Studio to marketing and enablement teams that have tested AI workflows but cannot reliably control which context those workflows use.

## Goals

- Show that governed Documents and Collections solve different retrieval problems.
- Drive demo requests from teams responsible for marketing knowledge operations.
- Demonstrate MCP tool testing as a way to inspect what agents can access.

## Primary Message

AI quality improves when teams govern reusable context instead of asking every user to attach a new pile of files.
