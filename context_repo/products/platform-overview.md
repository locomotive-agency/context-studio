# Context Studio Platform Overview

Context Studio is a web-based context management system for AI-assisted marketing work.

## Core Capabilities

- Governed Documents stored as version-controlled Markdown records.
- Scope hierarchy for company, brand, product, audience, campaign, and market context.
- Folder-level metadata inheritance with document-level overrides.
- Searchable Collections for raw supporting sources.
- MCP tools that let AI agents list folders, inspect indexes, retrieve metadata, read selected Documents, search surfaced Collections, and validate the repository.

## Boundaries

Context Studio does not make every document semantic-searchable. Semantic retrieval applies to Collections. Governed Documents are selected by type, scope, status, and validity.
