# Customer Profile Software

**Source:** [https://www.zoominfo.com/features/ideal-customer](https://www.zoominfo.com/features/ideal-customer)

## Summary

AI-powered ICP modeling that automatically defines ideal customer profiles based on past won and lost deals. Identifies patterns across contacts and companies to build a targeted account universe and score prospects against the model.


## Key Capabilities

- AI-generated ICP from historical win/loss analysis
- Automatic pattern detection across firmographic and behavioral attributes
- Account scoring against ICP model for prioritization
- Integration with ZoomInfo's 500M+ contact database for targeting
