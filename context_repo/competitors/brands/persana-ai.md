# Persana AI

## Sub-products

- [[persana-ai--nia-sdr|Persana AI Nia SDR]] — competes with GTM Workspace
- [[persana-ai--platform|Persana AI Platform]] — competes with GTM Workspace

## Summary

Persana AI is an agentic AI prospecting and GTM intelligence platform sold to SDR/BDR-heavy sales teams, GTM Engineers, and founder-led B2B operators that aggregates 100+ data providers, 75+ live buyer signals, and an AI SDR agent (Nia) into a single workspace with native multi-channel sequencing. It overlaps ZoomInfo's Data foundation, GTM Workspace (Nia is a directly competing seller-facing AI agent), GTM Studio (workflow / signal orchestration), Operations (CRM enrichment), and Enterprise APIs. The most material finding from this dossier is that on the same first-party domain Persana announced it is "joining forces with Rox" and will sunset the Persana platform "over the coming weeks" with a Rox migration path — meaning Persana is effectively winding down as a standalone competitor. No first-party Persana-vs-ZoomInfo comparison page surfaced in SERP.

## Product Offerings

### Persana AI prospecting platform (data + workflows)

- **What it does:** All-in-one AI sales prospecting workspace that unifies 100+ data sources, AI-powered enrichment, intent signals, and CRM automation in a single subscription so teams can build lists, enrich them, run plays, and sync to the rest of their stack (source: https://persana.ai/).
- **Key features:**
  - 100+ premium data providers in one subscription
  - Build ICP using firmographic, technographic, and behavioral signals
  - AI Research Agents that analyze websites, news, lead-score prospects, and surface personalization angles
  - Workflow construction with sophisticated triggers across data, enrichment, and outreach
  - Sync prospects and enriched data to CRM, email tools, Slack, and ad platforms
  - Native Chrome extension for list scraping and prospect summaries
  - Templates and saved prompts to reuse research workflows
- **Source URL:** https://persana.ai/

### Nia (Autopilot AI SDR agent)

- **What it does:** 24/7 AI Autopilot SDR agent that automates up to 90% of the sales development process, drawing on 1B+ global contacts across 75+ data sources to find buyers, multi-channel engage them, and replicate top-rep playbooks (source: https://persana.ai/autopilot/nia).
- **Key features:**
  - Automates up to 90% of SDR/BDR workflow
  - Access to over 1 billion global contacts and 200 million companies across 75+ data sources
  - Multi-channel engagement (email plus additional channels)
  - 24/7 automated outreach
  - AI-driven personalization at scale using firmographic, technographic, and behavioral data
  - Custom Playbooks that replicate the workflows of top-performing reps
  - Real-time signal capture for outreach timing
- **Source URL:** https://persana.ai/autopilot/nia

### AI Signals (real-time intent layer)

- **What it does:** Tracks 75+ live buyer signals — funding news, hiring trends, G2 reviews, website visits, keyword intent, technographics, social listening — and triggers downstream workflows when accounts or contacts change state (source: https://persana.ai/features/ai-signals).
- **Key features:**
  - Job Change Tracker for retaining and recapturing key contacts
  - Hiring Trends Analyzer / actively-hiring company spotlights
  - Technographics for lead segmentation
  - Website Visit and Keyword Intent capture
  - Funding news and competitor info tracking
  - Social listening across G2 reviews and other public sources
- **Source URL:** https://persana.ai/features/ai-signals

### AI Sequencer (native multi-channel outreach)

- **What it does:** Persana's inbuilt enterprise-grade sequencer that runs multi-channel, multi-touch outreach natively in the platform with built-in deliverability infrastructure, signal-triggered timing, and AI-generated dynamic copy (source: https://persana.ai/ai-sequencer).
- **Key features:**
  - Native email sending with built-in warmup
  - Multi-channel, multi-touch sequences with follow-up logic
  - Signal-triggered timing using real-time data signals
  - AI-generated, hyper-personalized copy based on company growth, job changes, funding, and tech stack
  - Customizable tone and dynamic personalization
  - Built-in HubSpot and Salesforce integrations from the sequencer
  - Unified analytics and reporting dashboard
  - Optional integration with external tools (Instantly, Outreach.io, Salesloft) instead of native sending
- **Source URL:** https://persana.ai/ai-sequencer

### Full Contact Data API

- **What it does:** Developer API exposing Persana's underlying person and company data plus enrichment, intent, and growth lookups for programmatic integration into customer systems (source: https://persana.ai/full-contact-api).
- **Key features:**
  - Person Profile Search by name, title, company, or keywords (returns roles, work history, social links)
  - Company Profile Search by website domain, LinkedIn URL, name, domain, or industry
  - Waterfall Email Lookup API for verified email addresses
  - Phone Number Lookup API
  - Company Growth API tracking employee size over time
  - Intent Signals & Job Changes API
- **Source URL:** https://persana.ai/full-contact-api

## Integrations & Ecosystem

- **CRMs:** HubSpot (native two-way sync), Salesforce (native, 1-click enrichment in Salesforce), Microsoft Dynamics 365 (per Slashdot integrations row) (source: https://persana.ai/integrations).
- **Sales engagement / outreach:** Outreach.io, Salesloft (listed as "Salesoft" on the integrations page), Instantly.ai, Smartlead, Lemlist, Heyreach, La Growth Machine, Woodpecker, Sendspark (source: https://persana.ai/integrations).
- **Data / enrichment:** 100+ data providers via Persana's marketplace (homepage); specific named partners on the integrations page include Datagma, Contactout, PredictLeads, Prospeo, LeadMagic, and Icypeas. Apollo.io is listed as a sync target ("Add Leads to Apollo Sequences"), RB2B for visitor identification, and Apify for web-scrape actors (source: https://persana.ai/integrations).
- **Other notable integrations:** ChatGPT and Claude (used as AI enrichment actions inside Persana workflows), Google Search (large-scale Google searches as a workflow node), GitHub (find stargazers as a signal source), Gmail (native email sending), Find Local Businesses (Google Places), Slack (alerts / notifications referenced on homepage), and the Full Contact Data API (Persana's developer API surface) (source: https://persana.ai/integrations, https://persana.ai/full-contact-api).

## Differentiators (vs the broader category)

- Bundles 100+ data providers and 75+ live buyer signals into a single subscription rather than charging per provider — homepage frames this as "one affordable subscription. No other subscriptions needed" (source: https://persana.ai/).
- Nia AI SDR agent positioned as a packaged AI-agent SKU automating up to 90% of SDR work using 1B+ contacts across 75+ data sources — not just an enrichment platform but a full autopilot agent (source: https://persana.ai/autopilot/nia).
- Native AI Sequencer with built-in warmup and deliverability infrastructure — sequencing is part of the platform rather than an add-on integration (source: https://persana.ai/ai-sequencer).
- Custom Playbooks that replicate top-rep workflows so the AI agent executes them at scale (source: https://persana.ai/autopilot/nia).
- Full Contact Data API with waterfall email lookup, phone lookup, and company-growth time-series — exposes the underlying data layer for programmatic use (source: https://persana.ai/full-contact-api).

## Crossover With ZoomInfo

This is the load-bearing section. Compare the competitor against the brand's positioning in `CONTEXT.md`.

- **Direct overlap:**
  - **Data foundation** — Persana's 1B+ global contacts, 200M companies, 75+ data sources, and explicit waterfall email + phone lookup API are the same building blocks ZoomInfo's 500M-contact / 100M-company / 30K-technographic data foundation sells (source: https://persana.ai/autopilot/nia, https://persana.ai/full-contact-api).
  - **GTM Workspace (seller front-end)** — Nia is sold as a 24/7 SDR/BDR agent that finds buyers, runs multi-channel outreach, and replicates top-rep Playbooks — a directly competing seller-facing surface to ZoomInfo's GTM Workspace AI-drafted outreach (source: https://persana.ai/autopilot/nia).
  - **GTM Studio (audience building, play orchestration)** — Persana's homepage 4-step framing (Build ICP → Track Live Signals → Deploy AI Agents → Connect+Convert) and "Workflows that drive revenue" pitch competes directly with GTM Studio's natural-language audience and play-orchestration use case (source: https://persana.ai/).
  - **APIs & MCP** — Persana exposes a developer API for person and company enrichment, waterfall email, phone lookup, and company-growth time-series — overlapping ZoomInfo's Enterprise API access lane (Data, Marketing, Platform APIs) (source: https://persana.ai/full-contact-api).
  - **ZoomInfo Operations (CRM data quality + routing)** — Persana's integrations page exposes native HubSpot two-way sync and Salesforce 360-degree integration; CRM enrichment is a stated use case (source: https://persana.ai/integrations).
- **Adjacent overlap:**
  - **ZoomInfo Sales (classic prospecting UI) and the broader sales-engagement category** — Persana ships a native AI Sequencer plus connectors to Outreach, Salesloft, Instantly, Smartlead, Lemlist, Heyreach, La Growth Machine, Woodpecker, and Sendspark — adjacent to ZoomInfo's classic Sales prospecting UI but with a different surface (composable workflows + native sequencer rather than a packaged seller app) (source: https://persana.ai/ai-sequencer, https://persana.ai/integrations).
- **No overlap:**
  - **Conversation intelligence (Chorus)** — no call-recording, transcription, or call-coaching product surfaced on any reviewed Persana page; Slashdot lists "Conversation Intelligence" features under ZoomInfo but not Persana (source: https://slashdot.org/software/comparison/Persana-AI-vs-ZoomInfo/).
  - **ZoomInfo Chat (data-powered website chat)** — no native website-chat product surfaced on Persana pages.
  - **GTM Context Graph reasoning across CRM + conversations + intent + behavioral signals** — Persana captures intent and behavioral signals but does not advertise a unified context graph that fuses customer first-party CRM + conversation + behavioral data the way ZoomInfo positions GTM Context Graph.
  - **MCP server / agent-builder ecosystem support** — Persana's API is a REST data API; no MCP server, MCP listing, or third-party agent-builder ecosystem play surfaced on reviewed pages.
- **Their pitch against ZoomInfo (if found):** _No public comparison page surfaced in SERP results._ The only Persana-vs-ZoomInfo content captured was a third-party Slashdot feature-comparison table that contains generic vendor descriptions and no Persana-authored pitch quote. A G2 reviewer cited on the homepage ranks Persana as a "Clay alternative" rather than positioning explicitly against ZoomInfo. See flag `vs_brand_missing`. Note also: Persana's first-party announcement that it is "joining forces with Rox" and sunsetting the Persana platform reframes any vs-pitch — the more important customer question becomes the Rox migration, not a head-to-head with ZoomInfo (source: https://persana.ai/blogs/exciting-news-our-next-chapter).
