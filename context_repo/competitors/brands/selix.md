# Selix

## Sub-products

- [[selix--autonomous-sdr|Selix Autonomous SDR]] — competes with GTM Workspace

## Summary

Selix is an agentic AI sales agent in the autonomous-outbound category, built and operated by SellScale and aimed at founders, lean SDR teams, and revenue leaders at B2B SaaS startups. It overlaps directly with ZoomInfo's GTM Workspace and legacy ZoomInfo Sales surfaces — Selix executes prospecting, research, copy, sequencing, and sending end-to-end from chat (Slack), email, and an in-app workspace, including its own outbound infrastructure (domains, warmup, deliverability). It does not own a B2B contact graph and explicitly consolidates ZoomInfo, Apollo, and Clearbit as upstream data sources, so the competitive threat is on the seller workflow / AI-agent layer rather than on data. Proof depth is thin (no G2/Capterra reviews, no published case studies, pricing not on first-party pages).

## Product Offerings

### Selix (AI sales agent)

- **What it does:** An end-to-end agentic AI "sales operator" that takes a natural-language campaign objective and autonomously executes outbound — pulling prospects, researching, writing copy, sequencing, sending, and reporting — through chat (Slack), email, and an in-app workspace.
- **Key features:**
  - Language Interface Experience (LIX) — chat with Selix in-app, in Slack, or by forwarding emails to selix@sellscale.com
  - Multi-agent architecture: Supervisor / Worker / Secretary roles coordinate tasks (homepage diagram)
  - Built-in tool library: Segment Creator, Market Map Maker, ICP Builder, AI Researcher, Sequence Generator, Company List Puller, Asset Ingestor, Top Asset Analyzer
  - Memory "built in" — Selix remembers user priorities and acts on them across sessions
  - Iteration engine that continually tests strategies (e.g. coffee chat vs. batch vs. InMail) so the winning play surfaces — "Iteration > Volume"
  - "Hallucination-safe" production-grade architecture (homepage claim)
  - Slack-native and email-native command surfaces — does not require a custom UI to issue work
  - End-to-end task chain on the homepage demo: pull prospects → add research → ingest assets → write sequence → launch campaign
- **Source URL:** https://selix.ai/

### Selix tool sub-modules (homepage tool grid)

- **What it does:** Selix exposes the discrete tools its agents call — these are not separate SKUs but functional modules of the single agent product.
- **Key features:**
  - Segment Creator — builds ICP segments from CSV input or chat
  - Market Map Maker — competitive / market mapping
  - ICP Builder — defines ideal customer profile
  - AI Researcher — researches prospects and accounts
  - Sequence Generator — drafts outreach sequences
  - Company List Puller — pulls company lists
  - Asset Ingestor — ingests CSVs, conference attendee lists, marketing collateral
  - Top Asset Analyzer — analyzes which content / assets drive replies
- **Source URL:** https://selix.ai/

### Outbound Infrastructure (sending stack)

- **What it does:** Selix provisions and manages email-sending infrastructure (domain registration, mailbox warmup, deliverability / spam monitoring) so agents can send at scale without a separate stack.
- **Key features:**
  - Email domain registering and scaling (set up new addresses for outreach) (source: https://woodpecker.co/blog/sellscale/)
  - Spam monitoring (bounce-rate / complaint tracking) (source: https://woodpecker.co/blog/sellscale/)
  - Multi-inbox management — homepage tagline "manage all inboxes + respond quick"
  - Replaces an SDR's typical 10-tool stack (homepage claim: "Average number of tools needed for an SDR. All included with Selix out-of-box.")
- **Source URL:** https://selix.ai/

### Data & enrichment layer

- **What it does:** Pulls and enriches prospect data from external sources to feed campaigns; not a proprietary B2B database — Selix consolidates other vendors' data inside the agent workflow.
- **Key features:**
  - Data enrichment from LinkedIn Sales Navigator, startup databases, and other web sources (source: https://woodpecker.co/blog/sellscale/)
  - Homepage "Consolidate your GTM stack" visual lists ZoomInfo, Apollo, Clearbit, ChatGPT, 6sense, HubSpot, Instantly, Smartlead, Mailstack, Fastmail as tools Selix consolidates
  - AI researcher writes personalized copy from enriched data
- **Source URL:** https://selix.ai/

## Differentiators (vs the broader category)

- Agentic, not dashboard-first — Selix is positioned as a "single-person hire" that performs outreach work rather than a tool an SDR operates (source: https://www.pulsrev.com/tools/selix)
- Slack- and email-native command surfaces (forward an email to selix@sellscale.com or chat in Slack and Selix executes) (source: https://selix.ai/)
- Bundles the full SDR tool stack — including sending infrastructure (domains, warmup, spam monitoring) — out of the box, replacing an estimated 10-tool stack (source: https://selix.ai/)
- "Anti-sales sales tool" positioning aimed at non-GTM-engineers and founders — explicit anti-positioning vs. tool-heavy GTM stacks like Clay (source: https://selix.ai/)
- Multi-agent architecture (Supervisor / Worker / Secretary roles) marketed as "hallucination-safe" and production-ready (source: https://selix.ai/)
- Iteration engine continually tests outreach strategies (coffee chat vs. batch vs. InMail) so the winning play surfaces — "Iteration > Volume" framing (source: https://selix.ai/)

## Crossover With ZoomInfo

This is the load-bearing section. Compared against ZoomInfo's three pillars (Data foundation + GTM Context Graph + Universal Access via APIs/MCP, GTM Workspace, GTM Studio).

- **Direct overlap:**
  - **GTM Workspace** (ZoomInfo's seller front-end, the next-gen successor to ZoomInfo Sales) — Selix targets the same workflow: prioritized accounts, AI-drafted outreach, and execution from a single chat surface. Selix's homepage demo (pull prospects → research → write sequence → launch campaign) is the same job-to-be-done as GTM Workspace. (source: https://selix.ai/)
  - **ZoomInfo Sales** (legacy seller prospecting UI + AI assist) — Selix's tool grid (Segment Creator, ICP Builder, AI Researcher, Sequence Generator, Company List Puller) maps directly to prospect-finding and outreach-drafting jobs ZoomInfo Sales has historically owned. (source: https://selix.ai/)
- **Adjacent overlap:**
  - **ZoomInfo Data foundation** — Selix is a downstream consumer, not a substitute. Selix's homepage shows ZoomInfo as one of the data sources it "consolidates." Selix has no proprietary contact / company graph. (source: https://selix.ai/)
  - **ZoomInfo Operations** (CRM data quality / routing) — Selix touches HubSpot and inbox management to handle replies but does not claim CRM data quality, routing, or RevOps workflows. Adjacency, not substitution. (source: https://selix.ai/)
- **No overlap:**
  - Proprietary B2B contact / company database — Selix has no underlying graph; explicitly an execution agent over other vendors' data
  - Conversation intelligence (Chorus equivalent) — no call recording / transcription / coaching surface on any reviewed page; Selix is upstream of the meeting
  - Marketing / ABM platform (ZoomInfo Marketing, GTM Studio) — Selix is outbound sales-only; no display ads, ABM orchestration, intent-based audience activation, or marketer-facing surface on reviewed pages
  - Intent data / technographics / firmographics as a feed — Selix consumes intent / tech signals via partners (e.g. 6sense logo) but does not sell them
  - APIs / MCP for agent builders — no public API, MCP server, or developer surface advertised; Selix is an end-product agent, not infrastructure for other agents
  - Email-sending infrastructure (Selix bundles it; ZoomInfo does not) — Selix includes domain registration, warmup, and spam / deliverability monitoring (source: https://woodpecker.co/blog/sellscale/); ZoomInfo's GTM platform does not, sellers integrate Outreach / Salesloft / Smartlead instead
- **Their pitch against ZoomInfo (if found):** _No public comparison page surfaced in SERP results._ The SERP query "Selix AI vs ZoomInfo" returned only third-party comparisons (Seamless vs ZoomInfo, Apollo vs ZoomInfo, Reddit threads) — no Selix-authored vs-page exists. ZoomInfo does appear as a logo on selix.ai's "Consolidate your GTM stack" visual, framing ZoomInfo as one of ~10 SDR tools Selix replaces in aggregate.
