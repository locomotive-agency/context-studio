# People.ai — MCP Integration

**Competes with:** APIs & MCP (direct overlap on the AI-tool integration / data-access channel)
**Spotlight rank:** 4

## What it does
Exposes People.ai's activity graph and revenue answers to external AI agents and tools via Model Context Protocol so the same data is available wherever the user works (web app, CRM, or third-party AI client).

## Spotlight when
The article compares MCP servers across the GTM space — People.ai's February 2026 MCP launch is one of the few revenue-intelligence-side counter-examples to ZoomInfo's data-side MCP.

## Cite
- Key features: MCP server announced Feb 2026, activity data exposed to AI tools, multi-surface answer access, agentic-AI-era positioning
- Strengths: Early-mover MCP among RI vendors; pairs MCP with a deep first-party activity dataset
- Gaps: Enterprise-only access; narrower data scope than ZoomInfo MCP; thin developer documentation
- Proof quote: "Access answers in the web app, embedded in CRM, or via AI tool / MCP integration" — https://www.people.ai/
