# Voila Norbert — Voila Norbert API

**Competes with:** APIs & MCP (adjacent overlap — REST API for find/verify/enrich without an MCP server or agent layer)
**Spotlight rank:** 5

## What it does
REST API exposing the same find / verify / enrich functions for embedding into customer applications and workflows — free API key on signup with multi-language client libraries.

## Spotlight when
The article compares API-first email finder / enrichment services or developer-accessible alternatives to ZoomInfo's enterprise APIs. Lean on the free API key and three-primitive scope.
