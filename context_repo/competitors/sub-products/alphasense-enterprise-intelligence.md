# AlphaSense — Enterprise Intelligence

**Competes with:** Universal Access (adjacent overlap on universal search across enterprise content)
**Spotlight rank:** 3

## What it does
Connects a customer's internal shared drives so memos, meeting notes, decks, and models become searchable alongside AlphaSense's external 500M+ document library, with one-day deployment and a private-cloud option.

## Spotlight when
Comparing "search across enterprise content" coverage. Enterprise Intelligence is the AlphaSense surface that touches universal-access territory — but only for documents, not CRM/intent/contact data.

## Cite
- Key features: Turn-key SharePoint/Box/Drive/OneDrive/Dropbox/S3 integrations, Direct upload + email forwarding, GenAI search/summarize/audit, One-day deployment, Private cloud, LLMs never trained on client data
- Strengths: Searches internal docs alongside AlphaSense external library; private cloud for regulated buyers; one-day deployment
- Gaps: Analyst search not GTM execution; documents not CRM/marketing data; no write-back to revenue tools
- Proof quote: "GenAI search/summarize/audit across internal and external content" — https://www.alpha-sense.com/platform/enterprise
