# AdRoll ABM — AdRoll ABM (formerly RollWorks)

**Competes with:** ZoomInfo Marketing (direct overlap on ABM workflows + ad orchestration; the first-party "AdRoll ABM vs ZoomInfo MarketingOS" comparison page exists)
**Spotlight rank:** 1

## What it does
Account-based marketing platform combining B2B audience segmentation and a cross-channel DSP in a single application — identify, target, engage, and measure high-fit accounts with native CTV and DOOH execution, real-time buyer signals, and one-click CRM sync.

## Spotlight when
The article compares ABM platforms head-to-head with ZoomInfo Marketing. AdRoll ABM has an authored vs-ZoomInfo page — strongest spotlight angle is "ABM intelligence + native ad execution in one app" vs. ZoomInfo's data-led + ad-module framing.

## Cite
- Key features: Buyer-insights engine (2.6B identities / 92M contacts), Command Center, InIQ + BidIQ, multi-touch cross-channel activation, predictive intent (G2/Bombora), closed-won attribution, native CTV/DOOH, one-click HubSpot/Salesforce
- Strengths: Native ABM + DSP in one app; native CTV/DOOH; 30-day onboarding
- Gaps: 92M contacts vs 260M+; no CI; pricing gated behind demo
