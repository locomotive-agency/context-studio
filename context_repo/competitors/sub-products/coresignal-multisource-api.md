# Coresignal — Coresignal Multi-Source API + Agentic Search

**Competes with:** APIs & MCP (direct overlap on the developer / agent-builder access lane)
**Spotlight rank:** 1

## What it does
Developer-grade REST API + Agentic Search API delivering company, employee, and jobs records with three processing tiers (Base / Clean / Multi-Source up to 500+ fields), Elasticsearch Query DSL filtering, and 176ms average response.

## Spotlight when
The article spotlights ZoomInfo's APIs & MCP lane against developer-first data providers, especially when the angle is API-only delivery, public credit pricing, or agentic search interfaces.

## Cite
- Key features: REST API with 176ms average response time, Three processing levels (Base/Clean/Multi-Source) up to 500+ fields, Elasticsearch Query DSL exposed, Bulk download up to 10K records per request, Webhooks for profile-change notifications, Agentic Search API
- Strengths: Elasticsearch Query DSL exposed; public credit-based pricing from free trial → $49; Base/Clean/Multi-Source processing tiers
- Gaps: No native MCP server; no CRM/sales-engagement connectors; no verified phone numbers/direct dials; smaller scope than ZoomInfo Data + Intent
