# HubSpot Sales Hub — Breeze Prospecting Agent

**Competes with:** GTM Workspace (direct overlap on the AI-agent layer for prospecting)
**Spotlight rank:** 2

## What it does
AI sales agent inside HubSpot's Breeze suite that finds accounts, sources buying committees from CRM + web + buying signals, and executes personalized outreach — metered via HubSpot Credits.

## Spotlight when
The article compares AI sales agents and Copilot-style assistants. Breeze's spotlight angle is the "agent embedded in Smart CRM" pitch versus ZoomInfo's GTM Workspace AI layer.

## Cite
- Key features: Account discovery from CRM + web + buying signals, Buying-committee sourcing, AI-personalized outreach, Credit-metered execution (100 credits per lead outreach), Apollo as one of the embedded data providers
- Strengths: Embedded in Smart CRM; available across paid tiers via HubSpot Credits; uses Apollo data as one of its sources
- Gaps: Beta product; relies on customer CRM + web + partner data rather than a proprietary B2B graph; per-outreach credit consumption scales with volume
- Proof quote: "AI agent that finds accounts, sources buying committees, and executes personalized outreach" — https://www.hubspot.com/products/sales
