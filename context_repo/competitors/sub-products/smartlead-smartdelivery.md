# Smartlead — SmartDelivery

**Competes with:** GTM Workspace (adjacent — deliverability-test layer for the seller-outbound surface)
**Spotlight rank:** 4

## What it does
Pre-send placement-test tool that scores inbox vs. spam vs. promotions placement and surfaces authentication / spam-trigger issues before a campaign launches.

## Spotlight when
Articles position deliverability tooling as a missing piece of GTM Workspace's seller surface or argue that pre-send placement testing should be table-stakes for outbound platforms.

## Cite
- Key features: Placement scoring across primary / promotions / spam; spam analysis with insights; test on non-connected accounts; SPF / DKIM / DMARC / CNAME checks
- Strengths: Pre-send placement test bundled in; auth + spam diagnostics in one tool; broader sample
- Gaps: Diagnostic only; no long-horizon reputation scoring; standalone tool
- Proof quote: "Score inbox / spam / promotions placement before campaigns send." — https://www.smartlead.ai/email-deliverability-test
