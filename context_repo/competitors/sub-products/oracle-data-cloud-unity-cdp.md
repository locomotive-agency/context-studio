# Oracle Data Cloud — Oracle Fusion Unity Data Platform

**Competes with:** GTM Context Graph (direct overlap on the reasoning/profile layer — Unity Dynamic Profile Engine + Intelligence Workbench mirror the "why now" fused-signal narrative)
**Spotlight rank:** 1

## What it does
Enterprise customer data platform that unifies first-party customer data from sales, service, marketing, finance, ERP, and back-office systems into a single governed customer profile, then activates AI-driven audiences, journeys, and agentic workflows.

## Spotlight when
The article frames a comparison between "enterprise CDP with system-of-record data" and ZoomInfo's GTM Context Graph — Unity's Forrester TEI + Gartner MQ Leader recognition and "enterprise truth not marketing truth" framing land hardest here.

## Cite
- Key features: Unified Enterprise Data + Identity Resolution, Dynamic Profile Engine, 27+ AI/ML models, 100+ behavioral attributes + 80+ scores, Real-time activation, OOB B2B/B2C/B2B2C models, Native AI agents, HIPAA/ISO 27001/SOC 2
- Strengths: System-of-record + behavioral fusion ("enterprise truth"); AI/ML native in Oracle DB; Forrester TEI 158% ROI + Gartner MQ Leader
- Gaps: Lists ZoomInfo as enrichment partner (consumer not substitute); heavy B2C industry coverage outside ZoomInfo scope; legacy ODC DMP wound down
- Proof quote: "Transform enterprise-wide customer data into actionable, trusted, AI-enabled customer profiles." — https://www.oracle.com/cx/customer-data-platform/
