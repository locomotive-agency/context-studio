# Highspot — Highspot Agents (Deal/GTM/Role Play)

**Competes with:** GTM Workspace (direct overlap on the agent layer for sellers + marketing/enablement)
**Spotlight rank:** 2

## What it does
AI agents that act on GTM signals — Deal Agent surfaces real-time signals across CRM, meetings, and seller activity to guide reps; GTM Agent spots gaps in content, training, and plays for marketing/enablement; Role Play reinforces skills.

## Spotlight when
The article addresses agent-layer competition or marketing-enablement automation. GTM Agent (marketer-side) and Deal Agent (seller-side) together are the head-to-head spotlight against GTM Workspace's agent surface.

- Key features: GTM Agent for marketers, Deal Agent for sellers, AI Role Play, Salesforce/Slack integrations, Native analytics with permissions
- Strengths: Marketer-side agent (GTM Agent); CRM+meeting+activity signal fusion; practice-to-performance loop
- Gaps: First-party signals only; no prospecting/discovery; no Operations-style hygiene
