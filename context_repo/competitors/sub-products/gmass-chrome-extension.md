# GMass — GMass Chrome Extension

**Competes with:** GTM Workspace (adjacent overlap on the seller send-side surface only)
**Spotlight rank:** 1

## What it does
Chrome extension that turns Gmail / Google Workspace into a mail-merge, cold-email, and mass-email platform — adds a send button and settings panel inside the Gmail Compose window with proprietary tech to break Gmail's daily send limits.

## Spotlight when
The article compares low-cost, individual-seller send tools to ZoomInfo's GTM Workspace seller surface. GMass's Gmail-native send story is the spotlight — call out that it's a sending pipe, not a data + intelligence platform.

## Cite
- Key features: Mass Gmail sending, Mail merge with Google Sheets, Unlimited follow-up sequences, A/B testing, REST API + Zapier
- Strengths: Gmail-native UX; public self-serve pricing; proprietary send-limit-breaking tech
- Gaps: No B2B data; no intent/CI; Gmail-only sender
