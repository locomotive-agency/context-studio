# Lusha — Lusha MCP

**Competes with:** APIs & MCP (direct overlap at the MCP / agent integration layer)
**Spotlight rank:** 1

## What it does
MCP server exposing Lusha's contact data to AI agents and LLMs — the second documented MCP from a Tier 1 ZoomInfo competitor (after Clay MCP).

## Spotlight when
The article is about MCP for B2B data, AI agents accessing GTM data, or developer/AI builder workflows. ZoomInfo MCP's exposure of verified data + GTM Context Graph reasoning + GTM Workspace context is the structural counter.

## Cite
- Key features: MCP server for AI agent integration, surfaces Lusha contact data to LLMs/agents, Claude/ChatGPT/Anthropic ecosystem compatibility
- Strengths: One of two Tier 1 competitors with MCP; AI-agent-native architecture
- Gaps: Surfaces only contact lookups (no intelligence/reasoning layer); narrower value than ZoomInfo MCP
- Proof quote: "Bring Lusha's B2B data to any AI agent via MCP." — https://www.lusha.com/blog/lusha-mcp-sales-use-cases/
