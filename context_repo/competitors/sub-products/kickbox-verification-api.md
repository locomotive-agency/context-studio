# Kickbox — Kickbox Email Verification API

**Competes with:** ZoomInfo Operations (adjacent overlap on form-fill / signup verification at point-of-capture; thinner enrichment surface)
**Spotlight rank:** 2

## What it does
REST API for real-time email verification at signup / form / POS / mobile capture, plus batch jobs up to 1M addresses, with a free sandbox mode and auto-recharge.

## Spotlight when
The article compares verification APIs for product / lifecycle teams — when free sandbox testing, did-you-mean correction, and 1M-row batch jobs are decision criteria.

## Cite
- Key features: Single + batch endpoints, deliverable/undeliverable/risky/unknown + reason, Sendex score, did-you-mean, sandbox, dashboard, auto-recharge, 95% accuracy
- Strengths: Free sandbox testing; did-you-mean at capture; 1M-address batch jobs
- Gaps: Validation only (no enrichment); no enterprise data-residency tier; no MCP/agent ecosystem
- Proof quote: "Real-time email verification at the point-of-capture (webforms, signup, mobile app, POS)" — https://kickbox.com/email-verification-api/
