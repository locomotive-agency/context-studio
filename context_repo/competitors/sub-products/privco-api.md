# PrivCo — Private Market Data API & Custom Datasets

**Competes with:** APIs & MCP (adjacent overlap — REST API but no MCP)

**Spotlight rank:** 2

## What it does
RESTful API plus analyst-built custom data exports delivering bulk private-company financial data into client systems via JSON or CSV — gated to PrivCo's Unlimited / Enterprise tier.

## Spotlight when
The article surveys data APIs across categories. PrivCo's API + custom-dataset combination is its most direct ZoomInfo APIs & MCP comparison point.

## Cite
- Key features: 146MM+ data points across 1,000 verticals, REST API with documented endpoints, bearer-token auth, JSON/CSV bulk exports, custom dataset engagements
- Strengths: Documented REST endpoints; custom datasets available; bulk delivery
- Gaps: No MCP server; gated to Unlimited / Enterprise only; no agent ecosystem
- Proof quote: "146MM+ data points across 1,000 verticals" — https://www.privco.com/product/api
