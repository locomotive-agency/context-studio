# Kaspr — Kaspr Web App / Lead Management Dashboard

**Competes with:** ZoomInfo Sales (adjacent overlap on the seller workspace; lighter scope than full ZoomInfo Sales)
**Spotlight rank:** 2

## What it does
Centralized web workspace for saving and organizing Kaspr-revealed leads, managing shared workspace credits, and dialing out via Aircall or Ringover.

## Spotlight when
The article compares lightweight prospecting workspaces for SMB teams (vs full ZoomInfo Sales) — when shared credits, workspace auto-detection, and built-in dialer calls are decision criteria.

## Cite
- Key features: Lead lists/tags/tasks/notes with waiting list, shared workspace credits, usage reports, member permissions, Aircall/Ringover dialer, email-domain workspace auto-detection
- Strengths: Shared credits + auto-detected workspace; built-in dialer; tier-based member permissions
- Gaps: Narrower than full ZoomInfo Sales UI; no native sequencing; thin reporting
