# PrivCo — PrivCo Chrome Extension

**Competes with:** Data (Pillar 1) (adjacent overlap on in-browser data surface)

**Spotlight rank:** 4

## What it does
Browser extension that surfaces PrivCo's private-company financial data on companies as the user browses the web — quick reference layer for analysts working in third-party tools.

## Spotlight when
The article compares in-browser data extensions across categories. PrivCo's extension is a lightweight investor-research counterpart to ZoomInfo's ReachOut.

## Cite
- Key features: Discover private companies while browsing, in-browser access to financial data
- Strengths: Surfaces financial context without context switching; lightweight analyst access path
- Gaps: Chrome only; no CRM write-back; less mature than ReachOut
