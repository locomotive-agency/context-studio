# Hunter — Hunter Email Verifier

**Competes with:** Data (Pillar 1) (adjacent — email validity is the deliverability layer that touches but doesn't substitute for ZoomInfo's full email-coverage stack)
**Spotlight rank:** 2

## What it does
Validates that any email address exists and is deliverable — syntax, MX, SMTP, plus proprietary accept-all (catch-all) verification — to reduce bounce rates and protect sender reputation.

## Spotlight when
The article compares email verification vendors (NeverBounce, ZeroBounce, Bouncer, Kickbox). Hunter's spotlight angle is "verification bundled with the same platform that finds the emails," reducing tool-hopping for SMB outreach teams.

## Cite
- Key features: Syntax / gibberish / disposable / MX / SMTP-level checks, Proprietary accept-all (catch-all) verification, Bulk Email Verifier, Free API access, Native HubSpot / Pipedrive integrations, 0.5 credit per verification
- Strengths: Proprietary catch-all verification with 15-20% accuracy uplift claim; failed verifications free; bundled with Email Finder
- Gaps: No B2B database behind it; point-tool scope; no ABM or intent overlay
