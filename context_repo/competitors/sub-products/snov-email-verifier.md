# Snov.io — Email Verifier

**Competes with:** Data (Pillar 1) (adjacent — verification slice of the data foundation)

## What it does
Multi-tier email verifier that cleans email lists before sending, with 7-tier checks, graylisting bypass, bulk verification, and a verification API.
