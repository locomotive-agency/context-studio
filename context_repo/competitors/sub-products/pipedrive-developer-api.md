# Pipedrive — Developer API

**Competes with:** APIs & MCP (adjacent overlap — REST API + marketplace, but no MCP)

**Spotlight rank:** 6

## What it does
Public REST API and developer hub for building integrations and custom workflows on Pipedrive data, plus a 500+ app marketplace covering chat, calendar, dialer, payments, and adjacent workflows.

## Spotlight when
The article surveys CRM APIs or frames the agent-builder ecosystem. Pipedrive is a clean contrast — open API and marketplace, but no MCP.

## Cite
- Key features: Public API Reference, plan-scaled tokens (up to 150K on Premium), 500+ marketplace apps, REST endpoints
- Strengths: Free public docs; published token quotas; AI-powered marketplace search
- Gaps: No MCP server; no agent-builder ecosystem; no developer-only tier
- Proof quote: "500+ Marketplace integrations with AI-powered natural-language search" — https://www.pipedrive.com/
