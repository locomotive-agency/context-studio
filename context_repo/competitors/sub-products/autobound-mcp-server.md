# Autobound — Autobound MCP Server

**Competes with:** APIs & MCP (direct overlap on the agent-builder access lane / MCP server surface)
**Spotlight rank:** 2

## What it does
Native Model Context Protocol server with seven tools (signal_search, company_enrich, contact_enrich, company_timeline, contact_timeline, signal_types, subscribe) that connects MCP-compatible clients (Claude Desktop, Claude Code, Cursor, Windsurf, ChatGPT) to 700+ B2B signals.

## Spotlight when
The article spotlights ZoomInfo's APIs & MCP lane / mcp.zoominfo.com server, especially when discussing how AI agent builders access B2B signal data.

## Cite
- Key features: Native MCP server connecting Claude Desktop/Code/Cursor/Windsurf/ChatGPT, Seven tools, Bulk enrichment up to 100 domains per call, OpenAI function-calling and Assistants API support, n8n HTTP-node automation, No SDK install required
- Strengths: First-class MCP support across the major clients; seven-tool surface designed for agentic workflows; OEM/white-label MCP deployment
- Gaps: Pricing gated; smaller signal corpus than ZoomInfo Data + Intent; no proprietary phone numbers; MCP scope is signals/enrichment only
