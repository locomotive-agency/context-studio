# Skrapp — API

**Competes with:** APIs & MCP (adjacent overlap on programmatic data access)
**Spotlight rank:** 5

## What it does
REST API for programmatic email finding and enrichment, available as an Enterprise add-on, with LinkedIn profile throughput up to 25 per second.

## Spotlight when
Articles compare ZoomInfo's APIs & MCP surface against lower-cost programmatic email/enrichment endpoints, or position SMB / agency engineering teams that need an API without a six-figure contract.

## Cite
- Key features: REST API; Enterprise tier add-on; up to 25 LinkedIn profiles per second
- Strengths: API behind a $262/mo plan; high LinkedIn throughput; public credit-priced consumption
- Gaps: No MCP server or agent ecosystem; gated to Enterprise tier; narrow scope vs ZoomInfo Data API
- Proof quote: "Process up to 25 LinkedIn profiles per second." — https://skrapp.io/compare/zoominfo-alternative
