# Autobound — Autobound Signal Intelligence (Insights Engine + REST API)

**Competes with:** Intent Data (direct overlap on the timestamped, ranked buyer-signal layer)
**Spotlight rank:** 1

## What it does
Real-time signal intelligence layer covering 700+ signal subtypes from 35+ sources (SEC filings, earnings, hiring, exec changes, social, web intent across 43K+ topics) over 250M+ contacts and 50M+ companies, delivered via REST API, GCS file push, or OEM licensing.

## Spotlight when
The article spotlights ZoomInfo's Intent Data product, especially when the angle is "signal intelligence vs static intent" or buyer-signal ranking by conversion likelihood.

## Cite
- Key features: 700+ signal subtypes across 35+ categories (SEC filings/earnings/hiring/exec/social/web-intent), 250M+ contacts and 50M+ companies coverage, Insight Relevance System, Sub-200ms REST API responses, GCS file push (CSV/JSON/Parquet), Web intent across 43K+ topics
- Strengths: Real-time timestamped source-attributed signals; reasoning layer ranks combined signals; OEM/white-label delivery
- Gaps: No phone number database; no org chart; no website visitor ID; no conversation intelligence; pricing gated
