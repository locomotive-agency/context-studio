# Validity DemandTools — Mass Modification

**Competes with:** ZoomInfo Operations (direct overlap on bulk CRM updates and reassignment)
**Spotlight rank:** 3

## What it does
Bulk edit, standardize, and reassign Salesforce records without exporting to spreadsheets — Modify, Tune, and Reassign modules cover record updates, grid editing, and ownership changes.

## Spotlight when
The article covers RevOps tooling for territory changes, mass field updates, or no-export Salesforce administration. Lean on Touch Record (fires Flows without value changes) and percentage-split reassignment as the spotlight angles.

## Cite
- Key features: Mass update with conditions/formulas/copy-paste, Object counts and field aggregates on related objects, Tune grid view with find-and-replace + pivots, Touch Record to fire record-triggered Flows, Reassign by criteria/percentage/parent owner, Preview + automatic restore files
- Strengths: Touch Record fires Flows without value changes; percentage-split / parent-owner reassignment; preview-before-publish with restore
- Gaps: Salesforce-only; no native field enrichment; requires CRM admin context
