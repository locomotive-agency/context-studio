# Talend Data Quality — Talend Data Fabric

**Competes with:** ZoomInfo Operations (adjacent — enterprise data fabric vs. GTM-resident data ops)
**Spotlight rank:** 3

## What it does
Unified data integration and governance platform combining ingestion, transformation, cataloging, quality, and stewardship across on-prem and cloud environments.

## Spotlight when
The article maps "horizontal data fabric" vs. "GTM-vertical data ops." Talend Data Fabric is the canonical horizontal example — useful when contrasting IT enterprise platforms with ZoomInfo's GTM focus.

- Key features: Smart Discovery, Data Preparation, Data Stewardship, App + API Integration, Reusable pipelines, RBAC + masking
- Strengths: Unified ingestion-to-stewardship; certification tracking; HL7/EDI support
- Gaps: No B2B data; no GTM surfaces; IT data-engineer buyer
