# Revenue Grid — Activity Capture (RG Capture)

**Competes with:** GTM Context Graph (direct overlap on the first-party signal substrate — Activity Capture is the Chorus/Workbounce-style context-capture engine for Salesforce shops)
**Spotlight rank:** 1

## What it does
Automatically captures emails, meetings, contacts, tasks, and attachments from Outlook and Gmail into Salesforce, with configurable rules and unlimited data retention.

## Spotlight when
The article addresses activity capture / first-party signal capture / EAC alternatives. Indefinite retention vs. EAC's 24-month cap and private-cloud deployment for FinServ are the strongest spotlight angles.

## Cite
- Key features: Auto-capture of emails/meetings/contacts/tasks, Configurable capture rules, Custom Salesforce object sync, Indefinite data retention, Cloud / private cloud / on-prem deployment
- Strengths: Indefinite retention vs. EAC 24-month cap; private cloud / on-prem deployment; SOC 2 + ISO 27001 + 100K+ user scale
- Gaps: Salesforce-only; no third-party data foundation; no standalone CI recorder
