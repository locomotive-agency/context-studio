# Avoma — Conversation Intelligence

**Competes with:** Chorus (direct overlap on coaching, scoring, and theme tracking)
**Spotlight rank:** 2

## What it does
Add-on that automates call scoring, real-time agent assist, and coaching analytics on top of the Avoma Meeting Assistant — the strongest direct head-to-head with Chorus on coaching/scorecards.

## Spotlight when
Comparing CI tools or coaching analytics. Avoma's real-time Answer Assistant with Answer Cards is unique among low-cost CI alternatives — call it out for any "live call assistance" framing.

- Key features: AI Scorecards (100% coverage), Real-time Answer Assistant + Answer Cards, AI coaching recommendations + scorecards, Custom Smart Trackers + Keyword Trackers, Talk-pattern insights, Smart phrase alerts, Top Themes
- Strengths: Real-time Answer Cards mid-call; auto-scoring 100% coverage; public $29/seat add-on pricing
- Gaps: Add-on layered on Meeting Assistant; no deal-side reasoning across CRM + intent + third-party data; smaller scorecard template library
