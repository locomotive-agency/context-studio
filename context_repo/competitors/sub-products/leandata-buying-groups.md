# LeanData — LeanData Buying Groups

**Competes with:** ZoomInfo Marketing (adjacent overlap on buying-group identification and orchestration without ZI's first-party intent and ad activation)
**Spotlight rank:** 3

## What it does
Buying Groups Blueprint add-on that identifies buying-group members and roles inside Salesforce, orchestrates multi-stakeholder journeys for target accounts, and coordinates SDR + Marketing activation — driven off the same FlowBuilder used for routing.

## Spotlight when
The article compares buying-group / multi-stakeholder ABM orchestration that lives inside Salesforce. LeanData's strongest angle: buying-group orchestration that inherits the customer's existing routing logic and audit trail rather than running in a separate ABM tool.

## Cite
- Key features: Buying group identification via scoring + signals, journey orchestration, engagement mapping, SDR+Marketing coordination, alerts for missing roles
- Strengths: Salesforce-native; same FlowBuilder used for routing; coordinates SDR + Marketing on one spine
- Gaps: No first-party intent layer; paid add-on; no media activation against the buying group
