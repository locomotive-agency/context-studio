# GetProspect — GetProspect Data Enrichment

**Competes with:** ZoomInfo Operations (adjacent overlap on bulk CSV/API enrichment for CRM data quality, but at SMB scale and without dedupe/routing layer)
**Spotlight rank:** 3

## What it does
Bulk and API-based data enrichment that adds emails, mobile numbers, and 30+ contact/company attributes to existing CRM or list data — with native sync to HubSpot, Salesforce, Pipedrive, and Zoho, plus a CSV/XLSX export up to 50K rows free.

## Spotlight when
The article compares CRM enrichment / hygiene tools at SMB scale — Enrichment's strongest spotlight angle is REST API + native CRM sync inside a freemium price band.

- Strengths: Contact + company enrichment in one product; REST API + native CRM sync at SMB price point; EU/GDPR-compliant
