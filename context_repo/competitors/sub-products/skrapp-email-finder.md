# Skrapp — Email Finder

**Competes with:** Data (Pillar 1) (direct overlap on the verified-business-email primitive)
**Spotlight rank:** 1

## What it does
Finds verified professional email addresses by name and company using web crawling, pattern recognition, and real-time verification, returning Valid / Catch-all status against a 200M-contact / 40M-company database refreshed daily.

## Spotlight when
Articles compare ZoomInfo to lower-cost email-finder alternatives or position SMB/freelancer access to verified emails without an enterprise contract — Skrapp's transparent credit pricing is the strongest spotlight angle.

## Cite
- Key features: 200M+ contacts and 40M+ companies, 92% search success rate, 97% verification accuracy, daily database refresh, web crawling + pattern recognition + real-time verification, Valid / Catch-all status returned
- Strengths: Free 50-credit/month tier; public per-credit pricing far below ZoomInfo Basic; Fair Credit Policy
- Gaps: Email-only (no phone / direct dial); much smaller than ZoomInfo's data scale; no technographic or intent depth
- Proof quote: "Find and enrich leads. Grow revenue." — https://skrapp.io/
