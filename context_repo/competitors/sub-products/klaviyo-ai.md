# Klaviyo — Klaviyo AI (K:AI)

**Competes with:** APIs & MCP (adjacent overlap on the MCP server / agent ecosystem direction; consumer-data scoped vs ZI's GTM-data scope)
**Spotlight rank:** 3

## What it does
Klaviyo's AI surface — Marketing Agent (campaign generation), Customer Agent (autonomous shopper resolution), Composer (prompt-to-campaign), Image Remix, channel affinity, plus a Klaviyo MCP Server, ChatGPT app, and Claude Connector for agentic ecosystem integration.

## Spotlight when
The article documents agentic / MCP ecosystem moves across vendors, or contrasts Klaviyo's consumer-data MCP with ZoomInfo's GTM-data MCP server.

- Strengths: MCP Server with agentic ecosystem; AI spanning marketing + service; 65% autonomous resolution claim
