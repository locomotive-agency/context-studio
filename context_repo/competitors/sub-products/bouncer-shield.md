# Bouncer — Bouncer Shield

**Competes with:** FormComplete (adjacent — real-time form protection vs ZI's form enrichment)

**Spotlight rank:** 3

## What it does
No-code real-time email-verification widget for web forms that blocks invalid, malicious, or fraudulent emails (and IPs) at the moment of entry.

## Spotlight when
The article frames sign-up form protection, anti-fraud, or no-code marketer-deployable form tooling. Bouncer Shield's "paste a script, no developer required" angle is the spotlight.

## Cite
- Key features: Drop-in JS snippet; block by email + IP; multiple shields per account; styling/CSS/branding controls; volume-tiered subscription
- Strengths: No-code real-time protection; block by email and IP; per-domain config
- Gaps: Form-protection only; no enrichment; no native CRM integration
