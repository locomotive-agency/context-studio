# IntentData.io — Prospect Data Integrations

**Competes with:** ZoomInfo Operations (adjacent overlap — distribution layer that pushes intent + contact records into the customer's CRM/MAP rather than competing with ZoomInfo's data-ops engine)
**Spotlight rank:** 3

## What it does
Pushes contact-level prospect records and intent signals into CRMs and marketing-automation platforms via a native Salesforce integration plus form-push delivery to Marketo, HubSpot, and any CRM/MAP with integrated forms.

## Spotlight when
The article covers how third-party intent data activates inside existing martech stacks (Salesforce / Marketo / HubSpot / Terminus) without a vendor's own workspace.

## Cite
- Key features: Native Salesforce, form-push to Marketo/HubSpot, selective import controls, Terminus/LinkedIn/Twitter/Drift/Triblio/Predictable Revenue process integrations, real-time BDR/AE/CS alerts, activation coaching
- Strengths: Selective import to control CRM costs; native + form-push delivery; bundled activation coaching
- Gaps: No API/MCP; no platform UI; Drift requires separate subscription
- Proof quote: "real-time delivery for BDR / AE / CS alerts" — https://www.intentdata.io/prospect-data
