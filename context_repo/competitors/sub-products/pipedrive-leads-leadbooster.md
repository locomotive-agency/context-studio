# Pipedrive — Sales Lead Management (Leads Inbox + LeadBooster)

**Competes with:** Data (Pillar 1) (adjacent overlap on contact data + lead capture)

**Spotlight rank:** 3

## What it does
Lead-capture, qualification, and routing layer — Leads Inbox holds pre-qualified leads outside the deal pipeline, while the LeadBooster add-on adds Chatbot, Live Chat, Web Forms, and a Prospector database.

## Spotlight when
The article compares lead-capture stacks for SMBs or frames "B2B database access" against scaled data vendors. The funnel-claim numbers (+93% pipeline) are the easiest spotlight hook.

## Cite
- Key features: Leads Inbox, Web Forms, Chatbot/Live Chat, Prospector global database, routing rules, lead labels
- Strengths: Capture-to-route inside the CRM; LeadBooster bundles chat + forms + prospecting; +93% pipeline claim
- Gaps: Prospector scale not quantified; no intent/technographics; Web Visitors gated behind add-on
- Proof quote: "Increase the number of deals in their pipeline by an average of 93%" — https://www.pipedrive.com/en/products/sales/leads
