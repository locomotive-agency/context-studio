# Salesforce Agentforce — Agentforce 360 Platform

**Competes with:** APIs & MCP (direct overlap on the agent-builder ecosystem and open-standards extensibility)
**Spotlight rank:** 3

## What it does
Underlying platform that powers Agentforce — combines metadata, governance, MuleSoft (integration), Data 360 (data), Einstein Trust Layer (security), MCP server, A2A, and AgentExchange on a single foundation for building and orchestrating agents.

## Spotlight when
The article surveys MCP servers and agent-builder platforms; Salesforce's MCP + MuleSoft Agent Fabric is the most enterprise-grade alternative to ZoomInfo's MCP/APIs for cross-platform agent orchestration.

## Cite
- Key features: MuleSoft Agent Fabric, Flow Orchestration, MCP/A2A/REST, 200+ templates, Data 360, Tableau, Apex/Prompt Builder
- Strengths: Native MCP, AgentExchange, MuleSoft Gartner MQ Leader, 426% ROI claim
- Gaps: Salesforce-centric metadata, no third-party B2B contact graph, paid CRM dependency
- Proof quote: "Connect your AI agents to external data and tools with Salesforce's Agentforce Model Content Protocol (MCP) servers." — https://www.salesforce.com/platform/agentforce-platform/
