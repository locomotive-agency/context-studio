# AiSDR — AiSDR Sales Autopilot

**Competes with:** GTM Workspace (direct overlap on the AI agent layer inside GTM Workspace — autonomous prospect-to-meeting workflow)
**Spotlight rank:** 1

## What it does
End-to-end AI sales agent that finds in-market buyers, researches each lead, writes personalized outreach across email/LinkedIn/AI-calls, qualifies replies, and books meetings — replacing what AiSDR's homepage diagram bills as nine point tools (including ZoomInfo).

## Spotlight when
The article compares ZoomInfo's AI agent layer inside GTM Workspace to autonomous AI-SDR agents, especially when discussing AI-generated multimedia outreach, DISC-tuned messaging, or "stack-consolidation" claims that name ZoomInfo directly.

## Cite
- Key features: Live AI lead search via plain-English ICP, Real-time deep per-lead research from LinkedIn/news/intent, Multichannel sequences with AI videos/memes/voice notes, AI Strategist GTM-plan generator, DISC personality analysis with auto-tuned tone, AI account scoring inside HubSpot/Salesforce
- Strengths: AI Strategist generates full GTM plans in ~20 minutes; DISC personality analysis tunes message tone; multimedia outreach (videos + voice notes) built into the sequence
- Gaps: No proprietary contact data depth vs ZoomInfo's 500M+; no conversation intelligence; no public API or MCP server; pricing starts at $900/mo quarterly (not SMB self-serve)
