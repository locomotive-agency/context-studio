# Lyzr — Lyzr Agent Platform

**Competes with:** GTM Workspace (direct overlap on the AI agent execution layer for sales — competes with the AI agent layer inside GTM Workspace)
**Spotlight rank:** 1

## What it does
Enterprise AI agent platform that bundles control plane, simulation engine, governance, and reliability infrastructure plus 100+ pre-built blueprints (including AI SDR, AI Phone Dialer, LinkedIn Outreach, AI Deal Nurturer) for production-grade agent deployment.

## Spotlight when
The article compares enterprise agent platforms vs sales-intelligence tools — Lyzr is the spotlight when the buyer is a CIO/COO/CAIO building cross-functional agents (sales + HR + finance + ops) with on-prem deployment and BYOM as table stakes.

## Cite
- Key features: AI SDR blueprint with multi-channel outreach (email/LinkedIn/phone), AgentMesh event-driven multi-agent orchestration, AI Agents for CRM Automation across Salesforce/HubSpot/Zoho, 100+ pre-built blueprints across HR/Sales/Marketing/Support/Banking/Insurance/IT/Procurement/VC, On-prem / private-cloud / VPC deployment with BYOM, Built-in Responsible AI + Hallucination Manager + Agent Eval
- Strengths: On-prem deployment with zero-data-access guarantee; model-agnostic BYOM; Forward Deployment AI Engineers
- Gaps: No proprietary B2B contact data; no GTM Context Graph equivalent; no public MCP server
