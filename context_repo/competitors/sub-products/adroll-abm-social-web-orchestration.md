# AdRoll ABM — Social & Web Orchestration

**Competes with:** ZoomInfo Marketing (none — walled-garden social orchestration outside ZI's footprint)
**Spotlight rank:** 4

## What it does
Cross-channel orchestration that lets a single AdRoll campaign run across the open web and the major walled-garden social platforms (Facebook, Instagram, TikTok, LinkedIn, Pinterest) from one DSP.

## Spotlight when
Rarely. Mention only if the article addresses unified social + web ad orchestration. Not a ZI competitor.

## Cite
- Key features: Cross-platform audience targeting (web + 5 social), Automation Builder, BidIQ, privacy-first identity graph
- Strengths: Walled-garden + open web in one console; privacy-first identity; no cross-advertiser sharing
- Gaps: Outside ZI footprint; not a B2B account graph; standalone tool
