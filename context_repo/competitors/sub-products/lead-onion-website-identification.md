# Lead Onion — Lead Onion Website Identification

**Competes with:** WebSights (direct overlap — both reveal companies + contacts visiting a customer's website)
**Spotlight rank:** 3

## What it does
Turns anonymous website visitors into named companies — and US-based contacts — in real time, using IP Flow's 1.4B+ B2B IP database for company match and Delivr.ai for person-level web-visit identification. 20k website lookups bundled into the pricing baseline; revealed visitors auto-route to CRM or cadence with topic/page-interest signals attached.

## Spotlight when
The article compares visitor identification against WebSights and the buyer values person-level (not just company-level) visitor reveal. Lead Onion's Delivr.ai-powered person-level reveal is the strongest spotlight angle.

## Cite
- Key features: Company-level visitor ID (IP Flow 1.4B+ IPs), Person-level web-visit ID (Delivr.ai, US), 20k website lookups baseline, Real-time CRM/cadence routing, Topic/page-interest signals
- Strengths: Person-level (US) web visit ID; IP Flow 1.4B+ B2B IPs; 20k lookups baseline
- Gaps: Person-level US-only; not a visitor-ID specialist (vs Lead Forensics); no analyst recognition
