# Braze — Braze Data Platform

**Competes with:** ZoomInfo Marketing (adjacent — first-party data unification, ICP mismatch)

**Spotlight rank:** 2

## What it does
Real-time first-party customer data layer that unifies, activates, and bidirectionally streams customer data into Braze and back out to analytics tools.

## Spotlight when
Useful as a real-time-streaming contrast to ZoomInfo's third-party-data-led model. Mostly relevant for ICP-mismatch B2C-vs-B2B framing.

## Cite
- Key features: Direct warehouse / digital / backend integrations; always-on real-time stream; bidirectional flow with analytics; internal AI updating live; first-party unification
- Strengths: Real-time stream vs batch; bidirectional flow; first-party-native
- Gaps: No third-party B2B data; B2C focus; not standalone
- Proof quote: "Always-on real-time stream that updates segments and personalization" — https://www.braze.com/product/overview
