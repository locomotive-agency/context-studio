# Kaspr — Kaspr API

**Competes with:** APIs & MCP (adjacent overlap on the contact-enrichment API surface; no MCP server analog)
**Spotlight rank:** 4

## What it does
REST API for contact enrichment, gated by paid tier with published per-tier rate limits, plus an "on request" approval step before keys are issued.

## Spotlight when
The article compares B2B-data APIs for developers / RevOps custom tools, especially when the buyer cares about European coverage but does not need MCP or an agent ecosystem.

## Cite
- Key features: 60 req/day Starter, 500 req/day Business, Unlimited Enterprise, on-request access on paid plans, REST enrichment endpoints
- Strengths: Tier-gated REST API with public limits; Unlimited on Enterprise; tied to Kaspr's European data
- Gaps: No MCP server; "upon request" friction; rate limits below ZoomInfo Enterprise
- Proof quote: "REST API (access on request) to enrich and sync contact data with custom tools" — https://www.kaspr.io/pricing
