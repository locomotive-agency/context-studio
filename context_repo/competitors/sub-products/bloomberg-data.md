# Bloomberg — Bloomberg Data (Enterprise Data Catalog)

**Competes with:** APIs & MCP (adjacent overlap — programmatic data access; different data scope)

**Spotlight rank:** 3

## What it does
Enterprise data feeds and management products that pipe Bloomberg market, reference, pricing, regulatory, sustainable-finance and event-driven data into client systems.

## Spotlight when
The article compares enterprise / API-based data delivery patterns. Bloomberg Data is a useful contrast point — same programmatic-access model, different data scope (market prices vs B2B GTM).

## Cite
- Key features: Enterprise Data Catalog (pricing/reference/regulatory/real-time/sustainable finance); Data License + Data License Plus; BLPAPI SDKs across 6 languages and 4 OSes; Excel Add-In; Cloud Access / Server API / iPaaS
- Strengths: Multi-language SDK depth; reference + pricing + regulatory data scope; direct enterprise pipes
- Gaps: Market data, not B2B GTM data; no MCP server; sales-gated enterprise licensing
- Proof quote: "BLPAPI SDKs for Python, Java, C/C++, .NET, Perl, Wolfram Language across Windows/Linux/macOS/Solaris" — https://www.bloomberg.com/professional/products/
