# UpLead — UpLead API

**Competes with:** ZoomInfo Sales (direct overlap on the developer-facing data API for enrichment and embedding — alternative entry point to UpLead's data foundation)
**Spotlight rank:** 3

## What it does
Real-time B2B data API exposing UpLead's underlying contact + firmographic + technographic data via Company, Person, Combined, Prospector, Logo, and Name-to-Domain endpoints — for CRM enrichment, automated prospecting, and embedding into custom apps.

## Spotlight when
The article compares developer-facing data APIs for embedding B2B data into custom workflows. UpLead API is the spotlight when the buyer wants endpoint-rich access (6+ endpoint types) with real-time verification at Pro-tier pricing.

## Cite
- Key features: Company API (lookup by domain or name), Person API (lookup by email or domain), Combined Person + Company API, Prospector API (filter by title/function/level/location), Company Logo API and Company-Name-to-Domain API, Full API access on Professional plan
- Strengths: Multiple endpoint types for varied embedding cases; API bundled with Pro tier; real-time verification on responses
- Gaps: Full API gated to Pro tier (custom pricing); no MCP server; no published SDK matrix
