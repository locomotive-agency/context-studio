# Adapt.io — Adapt Enrichment & API

**Competes with:** Data (Pillar 1) (direct overlap on the data-foundation enrichment + API delivery surface)
**Spotlight rank:** 2

## What it does
Keeps lead and CRM data fresh by appending and standardizing 50+ data attributes across CSV, CRM-sync, and REST-API workflows, with webhook-driven Refresh updates triggered by role and relocation events.

## Spotlight when
The article compares ZoomInfo's Data pillar / CRM Enrichment to budget B2B data providers, or specifically discusses real-time webhook-based enrichment as a workflow option.

## Cite
- Key features: CSV Enrichment, CRM Enrichment with Salesforce/HubSpot/Pipedrive/Zoho sync, Prospect API for verified email/phone, Enrich API for firmographic gap fills, Refresh API with webhook updates, 50+ enrichment attributes including direct dial and mobile number
- Strengths: Webhook-driven Refresh API for real-time CRM updates; built-in ZeroBounce verification on every plan; plug-and-play CRM Enrichment without engineering involvement
- Gaps: API access gated to Custom tier — no self-serve API motion; no data-warehouse destinations; no MCP server or agent integration; no conversation intelligence or first-party intent layer
- Proof quote: "Real-time data updates triggered by role changes or relocations" — https://www.adapt.io/platform/enrichment
