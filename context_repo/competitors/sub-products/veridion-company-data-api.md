# Veridion — Veridion Company Data API

**Competes with:** Data (Pillar 1) (direct overlap on the underlying B2B company-data foundation, with global coverage in under-served geographies and product-level catalog data as the wedge)
**Spotlight rank:** 1

## What it does
AI-curated global company-data API — Search API for programmatic supplier sourcing across 134M companies with complex filter language and confidence scores, plus Match & Enrich API for MDM / Supplier Master Record de-duplication and entity resolution.

## Spotlight when
The article highlights enterprise data foundations for procurement / supplier discovery / commercial insurance / ESG / TPRM use cases. Veridion is the spotlight when the buyer needs global supplier coverage, product-level catalog data, or private-company ESG data — none of which are ZoomInfo's GTM focus.

## Cite
- Key features: 134M companies tracked across 100+ countries with 320+ attributes per profile, Search API with complex filter language (and/or/match_expression operators) and confidence-scored results, Match & Enrich API with Legal & Operational match modes for entity resolution, Weekly refreshes across all data with >95% accuracy claim, NAICS / SIC / ISIC / NACE / NCCI / IBC classification taxonomies, 100% traceable datapoints with confidence scores per field
- Strengths: Coverage of under-served geographies (Asia, Middle East); product-level catalog data; ESG data on private companies via SASB
- Gaps: Pricing fully gated; no seller-facing UI; no MCP server or agent-marketplace
