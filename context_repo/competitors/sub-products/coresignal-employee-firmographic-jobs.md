# Coresignal — Coresignal Employee + Company + Jobs Datasets

**Competes with:** Data (Pillar 1) (direct overlap on the firmographic + people + jobs data foundation)
**Spotlight rank:** 2

## What it does
Bulk flat-file dataset delivery covering 75M+ companies (monthly refresh), 865M+ employees (continuous updates), and 461M+ jobs (daily refresh) — delivered as JSON/CSV/Parquet via web link, cloud-bucket push, or API retrieval.

## Spotlight when
The article spotlights ZoomInfo's Data pillar (especially employee/firmographic/jobs depth) for buyers building data-driven products or AI applications rather than consuming a sales UI.

## Cite
- Key features: Companies dataset 75M+ records (monthly full refresh), Employees dataset 865M+ continuously updated, Jobs dataset 461M+ postings with daily additions, Flat-file delivery as JSON/CSV/Parquet, Cloud-bucket push or web-link delivery, Pricing from $1000/yr
- Strengths: Employee dataset 865M+ is unusually deep; Jobs dataset depth for hiring-signal use cases; EWDCI ethical sourcing posture
- Gaps: No verified phone numbers/direct dials; no buyer-intent layer; no conversation intelligence; no CRM-resident enrichment; customer builds the UX
