# Close — Developer Platform & MCP

**Competes with:** APIs & MCP (direct overlap on the agent-builder distribution layer for CRM-of-record data)
**Spotlight rank:** 5

## What it does
REST API, webhooks, and an MCP server for connecting Close data to AI agents and external tools — ChatGPT and Claude integrations are listed natively.

## Spotlight when
The article compares MCP servers across CRMs and data tools, or "AI-agent-ready" SMB CRMs. Strongest spotlight when the buyer is building agents that need to read/write CRM-of-record data without enterprise gatekeeping.

## Cite
- Key features: REST API + webhooks, Close MCP server for AI models/agents, native ChatGPT + Claude integrations
- Strengths: Public MCP server unusual for SMB CRM; native ChatGPT/Claude; REST API + webhooks across paid tiers
- Gaps: CRM data payload only (no third-party B2B data); narrower than ZoomInfo's data-rich surface; Close-only object scope
