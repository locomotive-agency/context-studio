# People Data Labs — People Data Labs Person API

**Competes with:** Data (Pillar 1) (direct overlap on the underlying B2B person dataset, sold via API instead of a seller surface)
**Spotlight rank:** 1

## What it does
Comprehensive workforce profile data delivered via flexible APIs (Person Enrichment, Search, Identify, Bulk endpoints) and Data License Feeds, built for developers embedding people data into their own products — 1.5B person profiles with 140+ searchable attributes.

## Spotlight when
The article compares developer-facing data APIs / DaaS providers for embedding workforce data into the customer's own product. PDL Person API is the spotlight when the buyer is a builder who needs raw person data delivered programmatically (HR Tech, AI SDR vendors, investment research).

## Cite
- Key features: Person Enrichment API (match by name/location/email/phone/education/work/social), Person Search API (140+ searchable attributes), Person Identify API for entity resolution, Bulk Person Enrichment endpoint for scaled use, IP Enrichment API (IPv4/IPv6 → company/person metadata), 1.5B unique person profiles in the dataset
- Strengths: 140+ searchable attributes; pay-per-match billing with configurable strictness; multi-cloud Data License delivery
- Gaps: No seller UI; no CI or intent data; Free tier obfuscates contact fields
- Proof quote: "We build workforce data, so you don't have to." — https://www.peopledatalabs.com/
