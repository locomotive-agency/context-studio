# IntentData.io — Contextual Technographic Data (TechnoIntent)

**Competes with:** Data (Pillar 1) (direct overlap on technographics — competes with ZoomInfo's 30K+ technology coverage)
**Spotlight rank:** 2

## What it does
Blends technographic signals (web tags + parsed job postings) with contact-level intent data to indicate not just which software a company runs but whether they are actively researching alternatives — covers backend software beyond martech/salestech.

## Spotlight when
The article covers technographic data, competitive displacement plays, or renewal-window targeting (the 9–10-month research-mode flag is a strong displacement-marketing hook).

## Cite
- Key features: Tag-tracking + job-posting parsing, backend software coverage, Renewal Disruption / Knockout / Product Launch plays, market research output (penetration/churn/persona by geo+industry)
- Strengths: Tech + intent combined (research-mode flag); backend coverage; pre-built displacement plays
- Gaps: Smaller mapped tech surface vs ZoomInfo 30K+; no API/MCP; no usage telemetry
- Proof quote: "Renewal Disruption play: flags accounts in research mode 9–10 months into competitor contracts" — https://www.intentdata.io/contextual-technographic-data-incorporates-intent-signals
