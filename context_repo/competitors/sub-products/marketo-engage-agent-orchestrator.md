# Marketo Engage — Adobe Experience Platform Agent Orchestrator

**Competes with:** APIs & MCP (adjacent overlap — Agent Orchestrator is Adobe's AI-agent layer for cross-product orchestration; differs from ZoomInfo's MCP server in being Adobe-internal-agents-first)
**Spotlight rank:** 7

## What it does
Adobe's AI agent orchestration layer powering agents across Experience Platform — including Marketo Engage workflows for planning, audiences, content, journeys, experience management, and data/insights — with Adobe Marketing Agent accessible through ChatGPT Enterprise.

## Spotlight when
The article addresses agentic AI in marketing automation or AI-agent ecosystems. Agent Orchestrator's strongest spotlight: ChatGPT Enterprise as an entry point to Marketo workflows — but it's Adobe-product-focused rather than a developer-facing MCP.

## Cite
- Key features: Conversational AI Assistant prompts, reasoning engine, knowledge base for role-specific prompts, Agent Composer, coordinates Adobe agents (Brand Concierge / LLM Optimizer / etc.), Adobe Marketing Agent for ChatGPT Enterprise
- Strengths: ChatGPT Enterprise entry point to Marketo; cross-Adobe agent orchestration; Agent Composer for customization
- Gaps: No public MCP-style server; Adobe-product-focused agents (not 3rd-party builder ecosystem); limited interop outside Adobe Experience Cloud
