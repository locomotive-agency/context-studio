# Dreamdata — Dreamdata Data Platform

**Competes with:** GTM Context Graph (adjacent overlap on unified account-based data modeling)
**Spotlight rank:** 5

## What it does
Automated B2B CDP that collects, models, and unifies data from across the GTM tech stack into a clean, account-based 360° customer journey ready for AI activation, with direct warehouse access.

## Spotlight when
The article frames the data-foundation prerequisite for AI activation, ABM modeling, or open-warehouse customer data. Use as the lead when comparing CDP-style unified data layers vs ZI's Context Graph + first-party data foundation.

## Cite
- Key features: Automated tracking script + GTM integrations, account-based data model with dedup/auto-UTM, cookie + cookie-less tracking, Data Hub controls, BigQuery/Snowflake/Redshift/Azure/S3 access, SOC 2 Type II / GDPR / EU-default
- Strengths: Open warehouse access for custom queries/BI/reverse ETL; automated UTM mapping; cookie-less tracking
- Gaps: BYO contact data; no Chorus inputs; Free plan capped at 2 months history
- Proof quote: "AI is only as good as the data it's built on" — https://dreamdata.io/data-platform
