# Artisan — B2B Data

**Competes with:** Data (Pillar 1) (direct overlap on B2B contact + company + intent data)
**Spotlight rank:** 2

## What it does
Bundled contact database that powers Ava's outreach, with verified emails, direct dials, firmographics, technographics, and intent signals on 250M+ B2B professionals across 200+ countries.

## Spotlight when
The article compares contact databases or data scale across vendors. Artisan's angle is "bundled with the agent that uses it" — useful for a stack-consolidation framing, less so for pure data-quality comparisons where ZI's scale wins.

## Cite
- Key features: 250M+ B2B contacts in 200+ countries, Demographic + firmographic + technographic + intent, Waterfall enrichment across providers, Continuously verified, 2 credits/email and 10 credits/phone, Database included in platform
- Strengths: Bundled with autonomous outbound; waterfall across best-in-class providers; transparent credit pricing
- Gaps: Trails ZI scale; no DaaS/API-first enterprise packaging; no named multi-source verification frame
