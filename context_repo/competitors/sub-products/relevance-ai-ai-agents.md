# Relevance AI — AI Agents (Agent Builder)

**Competes with:** APIs & MCP (direct overlap on the developer / agent-builder universal-access surface)
**Spotlight rank:** 3

## What it does
No-code platform for building autonomous AI agents that automate GTM tasks, with a Programmatic GTM plugin for Claude Code, Cursor, and Codex (MCP-based).

## Spotlight when
The article covers MCP / agent-builder ecosystems or developer-grade GTM automation. The Programmatic GTM plugin is the sharpest spotlight angle vs. ZoomInfo's MCP positioning.

## Cite
- Key features: No-code Agent Builder, Any LLM / BYO LLM, "Invent" NL agent creation, Knowledge/memory store, Programmatic GTM MCP plugin
- Strengths: MCP + Claude Code/Cursor/Codex plugins; model-agnostic BYO LLM; marketplace templates
- Gaps: No proprietary data foundation; no GTM Context Graph reasoning layer; opaque Actions+Vendor Credits pricing at scale
