# Qualified — Qualified Agentic Marketing Platform

**Competes with:** GTM Workspace (direct overlap on the agentic-seller / orchestration layer)

**Spotlight rank:** 2

## What it does
Underlying platform that powers Piper — synthesizes GTM data, uses ML reasoning to plan and act on buyer engagement, and exposes Agent Spotlight (decision visibility) plus Agent Studio (training / content / guardrails / coaching) for ops teams.

## Spotlight when
The article digs into agent operations / training / guardrails — Agent Spotlight + Agent Studio are differentiators against simpler chatbot vendors.

## Cite
- Key features: Agent Spotlight, Agent Studio, Visitor 360 + waterfall enrichment, account segmentation / routing, reporting (native + Salesforce), Enterprise APIs, third-party intent
- Strengths: Decision-trace visibility; ongoing agent training operationalized; real-time GTM data synthesis
- Gaps: Enrichment relies on partner data; APIs gated to Enterprise; not a CRM of record
