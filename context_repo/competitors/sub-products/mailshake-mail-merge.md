# Mailshake — Mailshake Mail Merge

**Competes with:** GTM Workspace (adjacent overlap — table-stakes mail-merge functionality every sequencer ships, but framed as a hero feature inside Mailshake)
**Spotlight rank:** 4

## What it does
Sends one customized email to an entire group via field replacement tags, delivered as 1-1 emails rather than mass marketing blasts, with manual override mode to layer human personalization on top.

## Spotlight when
The article compares basic mail-merge / template-personalization features. Mailshake's strongest spotlight here: 1-1 send model + manual override — meets the "feels handwritten" test better than batch marketing tools.

## Cite
- Key features: CSV upload + per-column field replacement, field tags, manual override per recipient, scheduled delivery + follow-ups, native SF/HubSpot/Zapier
- Strengths: 1-1 email delivery; manual-override personalization; CRM-native
- Gaps: Table-stakes feature (parity with all sequencers); no AI personalization layered in; CSV-based list management
