# Voila Norbert — Enrich

**Competes with:** ZoomInfo Operations (adjacent overlap — narrow person-level enrichment vs ZoomInfo's full firmographic + technographic + intent layer)
**Spotlight rank:** 3

## What it does
Pay-as-you-go enrichment service that takes an email and returns the contact's job title, company, location, and social profiles — available via UI and via Enrichment API.

## Spotlight when
The article covers light-weight email-to-contact enrichment for SMB outbound or recruiter use cases. Lean on the pay-as-you-go model as a contrast to ZoomInfo's seat-based platform.

## Cite
- Key features: Returns job title/company/location/social profiles per email, Pay-as-you-go pricing tiered by list size, Available via UI and Enrichment API
- Strengths: PAYG with no minimum subscription; tiered per-email pricing; API-accessible
- Gaps: Title + location + social only — no firmographics/technographics/intent; no CRM-native workflow; $4 min
