# Acxiom — Real ID / Real Identity

**Competes with:** GTM Context Graph (adjacent — directionally similar plumbing applied to consumer identity rather than B2B account/contact reasoning)
**Spotlight rank:** 1

## What it does
Connected consumer identity solution recognizing and connecting people across online and offline touchpoints to power people-based marketing — Acxiom's flagship identity graph at 2.6B consumers globally.

## Spotlight when
The article frames identity-graph plumbing or first-/third-party stitching for marketing. Real ID is the strongest spotlight when the comparison is "B2C identity at scale" vs. ZoomInfo's B2B GTM Context Graph — call it adjacent, not direct.

## Cite
- Key features: 2.6B consumers / 260M US addressable, 5T+ records/month, 10K+ attributes, cookieless visitor recognition, 1.7x DSP reach, brand-controlled identity graph
- Strengths: Six-decade incumbent; cookieless recognition; native data-cloud + DSP integrations
- Gaps: B2C only; no MCP / agent surface; managed-service delivery
- Proof quote: "Real ID is built for how digitally native people live today" — https://www.acxiom.com/identity-resolution/real-id/
