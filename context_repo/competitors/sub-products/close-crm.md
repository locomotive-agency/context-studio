# Close — Close CRM (core platform)

**Competes with:** ZoomInfo Operations (adjacent overlap — Close handles native CRM data quality and routing, redundant for SMBs)
**Spotlight rank:** 1

## What it does
All-in-one sales CRM that combines lead/contact management, multi-channel communication, automation, and reporting in a single tool aimed at small and scaling sales teams.

## Spotlight when
The article compares SMB CRMs, "system of action" sales-team CRMs, or Close vs Salesforce/HubSpot/Pipedrive at the SMB tier. Strongest spotlight when the buyer is choosing a primary CRM, not augmenting one.

## Cite
- Key features: Lead/contact mgmt with custom objects, multi-pipeline tracking with Pipeline Guidance, Smart Views, no-code forms, multi-channel inbox, REST API + webhooks + event log API
- Strengths: Pipeline Guidance for stalled deals; native form builder; public REST API + webhooks on every paid tier
- Gaps: Not for teams >100 reps (own positioning); no third-party B2B contact database; no ABM/marketing
- Proof quote: "The Sales CRM That Does the Work For You" — https://close.com/
