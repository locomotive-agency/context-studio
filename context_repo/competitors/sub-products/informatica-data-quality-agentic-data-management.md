# Informatica Data Quality — Informatica Agentic Data Management

**Competes with:** GTM Context Graph (adjacent — both pitch trusted enterprise data as the foundation for AI agents, but Informatica is metadata-driven across all data while ZoomInfo's Context Graph is GTM-specific)
**Spotlight rank:** 4

## What it does
Use case framework that positions IDMC + CLAIRE as the trusted-data foundation for enterprise AI agents — including agent engineering, orchestration, and governance for both Informatica's own and third-party agents.

## Spotlight when
The article frames trusted-data foundations for AI. Agentic Data Management's spotlight angle is "metadata-grounded enterprise AI agents on any data," contrasted with ZoomInfo's "GTM-grounded AI agents on B2B data + signals."
