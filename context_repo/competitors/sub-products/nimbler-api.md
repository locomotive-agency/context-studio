# Nimbler — Nimbler API

**Competes with:** ZoomInfo Operations (direct overlap on CRM data quality + enrichment via API for the developer/RevOps consumer)
**Spotlight rank:** 2

## What it does
REST API + Datarade-packaged data products that expose Nimbler's underlying contact dataset for programmatic enrichment of CRM contacts (up to 50 attributes appended including mobile phone, personal email, job title, skills) and bulk file enrichment via the UI.

## Spotlight when
The article compares developer-facing data APIs for CRM enrichment workflows. Nimbler's API is the spotlight when the buyer wants packaged datasets via Datarade plus REST endpoints for programmatic enrichment.

## Cite
- Key features: REST API for B2B contact data + enrichment (JSON/XML/CSV/XLS), Bulk-file CSV enrichment via UI, API enrichment for programmatic workflows, Multiple delivery cadences (real-time/on-demand/scheduled), 5 years of historical data, CCPA and GDPR compliant data delivery
- Strengths: Up to 50 enrichment attributes via API; REST API + UI parity; Datarade subscription/yearly/one-off licensing
- Gaps: No MCP server or agent ecosystem; CRM connector integrations beyond SuiteCRM not confirmed; pricing range very wide
