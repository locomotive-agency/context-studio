# Aviso AI — MIKI Agentic AI Platform

**Competes with:** GTM Workspace (direct overlap on the AI-agent layer driving seller workflows)
**Spotlight rank:** 1

## What it does
Aviso's flagship agentic layer: real-time AI Avatars, 30+ out-of-the-box agentic workflows, 50+ task-based revenue agents, and a no-code GTM Agent Studio orchestrated by MIKI — billed as a tech-stack consolidator for the GTM motion.

## Spotlight when
Comparing AI agent platforms for revenue teams. MIKI is the canonical "full agentic stack" pitch — strongest spotlight when contrasting orchestrator depth or avatar/task-agent breadth against ZoomInfo's GTM Workspace agent layer.

## Cite
- Key features: MIKI orchestrator (NLP + CRM updates), Real-time AI Avatars, 30+ agentic workflows, 50+ task-based agents, No-Code GTM Agent Studio, CSM avatar + renewal cadences
- Strengths: Orchestrator + avatars + 50+ agents + no-code studio in one stack; role-specific avatars (BDR→CRO); MIKI writes back to CRM/deal stages
- Gaps: No third-party B2B data underneath; no MCP/external dev surface; revenue-execution-only (not ABM)
