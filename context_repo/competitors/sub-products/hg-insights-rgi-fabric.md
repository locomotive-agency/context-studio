# HG Insights — HG RGI Fabric

**Competes with:** Data (Pillar 1) (direct overlap on the data-foundation enrichment lane)
**Spotlight rank:** 2

## What it does
Underlying unified B2B data layer (firmographics, technographics, IT spend, buying centers, competitive, buyer intent) delivered via API, S3, and CRM connectors to power MDM/CRM enrichment and AI agents.

## Spotlight when
The article focuses on technographics, IT spend, or AI-buyer signals. RGI Fabric is HG's strongest substitute claim against ZoomInfo's data foundation — frame as "account strategy vs rep activation" per the third-party prospeo.io framing.

## Cite
- Key features: 20K+ technographics, 140+ category IT spend, Layered intent + TrustRadius 2nd-party, AI Maturity Index, REST API + S3 + Snowflake + MCP + Agent Builder
- Strengths: Bottom-up IT spend forecasting; behind-the-firewall detection; AI specialty datasets
- Gaps: Shallower contact/phone/email coverage; no CI; no demand-capture surfaces
