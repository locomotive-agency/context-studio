# ActiveCampaign — Integrations & MCP

**Competes with:** APIs & MCP (direct overlap on the agent-builder + MCP exposure surface)
**Spotlight rank:** 5

## What it does
1,000+ app marketplace plus a Model Context Protocol (MCP) server so AI tools like Claude and ChatGPT can act on the user's ActiveCampaign account, alongside a public developer API.

## Spotlight when
The article compares MCP servers across GTM tools. ActiveCampaign's MCP is a direct parallel to mcp.zoominfo.com/mcp — strongest spotlight is "MCP for marketing automation tools" but call out that AC's MCP exposes first-party data only.

## Cite
- Key features: 1,000+ integrations, Salesforce/Dynamics/Zendesk connectors, ecommerce stack, ad-platform syncs, Zapier/Make, Clay, MCP server, public API
- Strengths: Native MCP for Claude/ChatGPT; 1,000+ integrations; open developer API
- Gaps: First-party MCP surface only; no B2B graph access; limited GTM-engineering scope
- Proof quote: "Connect AI tools like Claude and ChatGPT directly to your ActiveCampaign account via our MCP server" — https://www.activecampaign.com/platform/active-intelligence
