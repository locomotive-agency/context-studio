# Jasper — API & MCP

**Competes with:** APIs & MCP (adjacent overlap — Jasper's MCP exposes content-agent capabilities; ZoomInfo's MCP exposes B2B data — distinct surface areas)
**Spotlight rank:** 5

## What it does
Programmatic access to Jasper from external systems — Business-plan-only API access, MCP support called out in top nav alongside REST API, plus native integrations across Slack, Google Drive, SharePoint, Workfront, Semrush, BigQuery, Webflow, and a Salesforce Beta connector.

## Spotlight when
The article covers the agent-builder / MCP-server ecosystem and contrasts content-MCP (Jasper) with data-MCP (ZoomInfo APIs & MCP). Useful for the developer/agent-builder ICP framing.
