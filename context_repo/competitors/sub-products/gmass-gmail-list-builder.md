# GMass — GMass Gmail List Builder

**Competes with:** GTM Workspace (adjacent overlap on audience-build slice)
**Spotlight rank:** 5

## What it does
Builds a mailing list from any Gmail search or label and opens a compose window pre-loaded with those addresses, no spreadsheet required. Auto-detects first names with fallbacks and supports recurring search-based campaigns.

## Spotlight when
The article highlights idiosyncratic outbound list-build motions (warm-list outreach, re-engagement of past Gmail contacts) — a niche where ZoomInfo's data-first model isn't the right tool.

## Cite
- Key features: Build from Gmail search, Build from Gmail label, First-name detection, Combine with Sheet list, Recurring search campaigns
- Strengths: No-spreadsheet list build from Gmail history; dynamic recurring searches; unique vs. ESPs
- Gaps: List quality depends on user's Gmail; no firmographic/technographic filters; no data enrichment
