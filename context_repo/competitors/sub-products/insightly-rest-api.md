# Insightly — Insightly REST API

**Competes with:** APIs & MCP (adjacent overlap — REST CRM access; not the AI-agent / MCP surface ZoomInfo ships)
**Spotlight rank:** 6

## What it does
REST + JSON API for embedding CRM functionality into custom applications and integrations — SDKs for Java, .NET, and Python plus Swagger documentation.

## Spotlight when
The article enumerates the developer/agent-builder ICP and contrasts CRM-only REST APIs with ZoomInfo's APIs + MCP server / agent ecosystem.

## Cite
- Key features: REST + JSON, Java/.NET/Python SDKs, web sandbox, Swagger, custom reporting, lead capture & sync, workflow triggers
- Strengths: Three first-party SDKs; sandbox for testing; Swagger auto-generation
- Gaps: No MCP / agent surface; CRM-only data; no agent-builder ecosystem
- Proof quote: "Third-generation REST + JSON API for embedding CRM functionality into custom applications and integrations" — https://www.insightly.com/api-for-developers/
