# Hunter — Hunter Data Platform & MCP

**Competes with:** APIs & MCP (direct overlap on programmatic access + MCP-server access lane)
**Spotlight rank:** 5

## What it does
API-only access to Hunter's email finding and verification services with pay-per-credit pricing, plus a remote MCP server that lets any LLM or agent call Hunter endpoints in natural language.

## Spotlight when
The article compares developer / agent-builder access to B2B data — programmatic APIs and MCP servers. Hunter Data Platform's spotlight angle is "remote MCP server + email-only API at credit-based pricing."

## Cite
- Key features: API-only access to Email Finder + Email Verifier, Pay-per-credit pricing (12-month credit validity), Bulk task support, Remote MCP server callable by any LLM/agent in natural language, Search credits and Verification credits as separate meters
- Strengths: Public remote MCP server (uncommon); pay-per-credit suits agent workloads; Hunter API exposed for both SDK and LLM callers
- Gaps: Email-only API surface; smaller dataset than ZoomInfo APIs & MCP; pay-per-credit costs scale with volume
- Proof quote: "Remote MCP server lets any LLM call Hunter API endpoints in natural language" — https://hunter.io/integrations
