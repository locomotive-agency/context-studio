# Klenty — Klenty SDRx / AI List Builder

**Competes with:** GTM Workspace (direct overlap on the AI SDR / natural-language list-building surface)
**Spotlight rank:** 5

## What it does
SDRx is Klenty's AI SDR agent that turns a natural-language ICP description into an ICP-matched account list, runs research across 150+ public data sources to surface a tailored value proposition per account, and pipes those insights directly into prospect fields for cadence personalization.

## Spotlight when
The article compares natural-language list-builder UIs and AI SDRs (e.g., 11x Alice, Apollo AI, Outreach AI Prospecting) against ZoomInfo's GTM Workspace AI agent layer for SMB/mid-market teams.

## Cite
- Key features: Natural-language ICP entry, 150+ public-source research, Per-account value proposition, Insights piped to prospect fields
- Strengths: Chat-style ICP entry; per-account value proposition; bundled with SEP
- Gaps: No verified B2B contact universe; narrower agent stack than GTM Workspace; no MCP/agent-builder API
