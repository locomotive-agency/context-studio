# Crayon — Crayon AI (Sparks + Crayon Answers)

**Competes with:** GTM Workspace (adjacent overlap on the AI-agent layer for revenue teams)
**Spotlight rank:** 1

## What it does
AI layer that aggregates competitor data, scores its importance, and turns it into seller-ready content and conversational answers across the Crayon stack — Sparks runs scheduled prompts; Crayon Answers serves Slack/Teams; the MCP server pipes validated content into ChatGPT, Claude, Copilot, Glean, Coveo, and Guru.

## Spotlight when
The article compares MCP servers / AI-agent layers across vendors. Crayon's strongest spotlight angle is "compete-content MCP server grounding ChatGPT/Claude/Copilot answers" — narrower payload than ZI's MCP but the same access pattern.
