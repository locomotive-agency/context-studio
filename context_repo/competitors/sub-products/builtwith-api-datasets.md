# BuiltWith — BuiltWith API & Datasets

**Competes with:** APIs & MCP (direct — programmatic data access with native MCP)

**Spotlight rank:** 4

## What it does
Programmatic access to BuiltWith technology data via per-domain API credits, bulk Firehose lookups, downloadable Parquet/CSV datasets, and a native Claude MCP connector.

## Spotlight when
The article frames developer / agent-builder data access, MCP-enabled workflows, or whole-internet technographic datasets. Native MCP + public PAYG pricing are the spotlight angles.

## Cite
- Key features: PAYG API ($99 / 2K credits, no expiry); Firehose ($315.45 / 5K sites); Datasets ($1,721 – $187K); Parquet/CSV for major warehouses; Claude MCP + AI Agents endpoint
- Strengths: Native MCP server; public PAYG pricing; whole-internet dataset delivery
- Gaps: Web-technology data only; no contact/firmographic API; steep top-end Datasets pricing
