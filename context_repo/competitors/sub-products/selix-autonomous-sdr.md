# Selix — Selix Autonomous SDR

**Competes with:** GTM Workspace (direct overlap on the seller-facing AI agent surface — competes with the AI agent layer inside GTM Workspace)
**Spotlight rank:** 1

## What it does
End-to-end agentic AI "sales operator" that takes a natural-language campaign objective and autonomously executes outbound — pulls prospects, researches, writes copy, sequences, sends, and reports — through chat (Slack), email, and an in-app workspace.

## Spotlight when
The article compares agentic AI SDR / autonomous outbound platforms aimed at founders or lean sales teams who want a "single-person hire" instead of a dashboard. Selix is the spotlight when the buyer is non-technical and prefers Slack-native command over a UI.

## Cite
- Key features: Language Interface Experience (LIX) — chat in-app, in Slack, or by forwarding emails, Multi-agent architecture (Supervisor / Worker / Secretary roles), Built-in tool library (Segment Creator, ICP Builder, AI Researcher, Sequence Generator, Company List Puller, Asset Ingestor, Top Asset Analyzer), Iteration engine that tests strategies (coffee chat vs batch vs InMail), Memory "built in" remembers user priorities across sessions, Bundled outbound infrastructure (domains, warmup, spam monitoring)
- Strengths: Slack- and email-native command surfaces; anti-sales / founder-friendly positioning; bundles SDR stack including sending infra
- Gaps: Pricing not published on first-party pages; no G2/Capterra reviews or named case studies; no proprietary contact graph
