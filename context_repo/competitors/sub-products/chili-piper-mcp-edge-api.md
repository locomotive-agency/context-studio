# Chili Piper — MCP Server & Edge API

**Competes with:** APIs & MCP (direct overlap on the agent-builder distribution layer)
**Spotlight rank:** 5

## What it does
Model Context Protocol server and full Edge API that let AI agents and developer tools query and act on Chili Piper — listing meetings, adjusting distributions, fetching slots, and booking — server-side with no human in the loop.

## Spotlight when
The article compares MCP servers / agent-builder distribution layers, programmatic scheduling, or APIs & MCP alternatives. Strongest spotlight when the buyer is building voice agents or AI SDRs that need to book meetings without UI.

## Cite
- Key features: MCP server for Claude Code/Cursor/OpenAI Codex, read + write access to org/users/distributions/routing/meetings, full booking flow via Edge API, Concierge/Handoff/ChiliCal endpoints
- Strengths: Server-side booking with no human in loop; voice-agent + AI-SDR use cases; RevOps automation via natural language
- Gaps: Scheduling/routing payload only (no contact/firmographic data); Edge API gated to Routing & Scheduling tier+; narrower than ZoomInfo MCP scope
