# Clearbit — HubSpot Breeze Intelligence

**Competes with:** Data (Pillar 1) — direct overlap; secondary overlap with ZoomInfo Operations + WebSights
**Spotlight rank:** 1

## What it does
The rebranded Clearbit data + intelligence layer inside HubSpot's Breeze platform — company/contact enrichment, visitor reveal, form shortening, all consumed via HubSpot Credits.

## Spotlight when
The article compares HubSpot-native data layers vs. external B2B data platforms. Frame Breeze Intelligence as HubSpot-bound, ZoomInfo as platform-agnostic verified data + intelligence.

- Key features: Company + contact enrichment via HubSpot Credits, Reveal-style visitor ID, form shortening with progressive profiling, native HubSpot integration
- Strengths: HubSpot bundling; unified Credits pricing
- Gaps: Smaller dataset; HubSpot-bound; standalone free tools sunset 2025-12-01
