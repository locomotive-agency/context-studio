# Fireflies — Fireflies MCP Server

**Competes with:** APIs & MCP (adjacent overlap — meeting-data MCP, not GTM-data MCP)
**Spotlight rank:** 6

## What it does
Model Context Protocol server that exposes Fireflies meeting insights to external AI tools (Claude, Devin, ChatGPT) — one-click connection to bring meeting insights into AI agent workflows.

## Spotlight when
The article covers MCP server adoption and agent-builder ecosystems. Fireflies MCP Server is the spotlight when the angle is "meeting-data MCP for agents" — distinct from ZoomInfo's GTM-data MCP.

## Cite
- Key features: One-click connection to Claude/Devin/ChatGPT, Bring meeting insights into AI agent workflows
- Strengths: One-click Claude/Devin/ChatGPT connection; API access in Free tier; MCP server bundled in
- Gaps: Meeting data only — not B2B data; thin documentation in dossier; no multi-tenant governance shown
- Proof quote: "Bring meeting insights into AI agent workflows" — https://fireflies.ai/
