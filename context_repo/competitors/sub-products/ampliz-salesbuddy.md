# Ampliz — Ampliz SalesBuddy (Chrome extension)

**Competes with:** ZoomInfo Sales (direct overlap on the in-page prospecting + Chrome extension workflow)
**Spotlight rank:** 3

## What it does
Chrome browser extension that surfaces company, contact, and contextual intelligence on the page the user is browsing — with a prospect-fit score against the user's defined ICP.

## Spotlight when
The article spotlights ZoomInfo Sales' in-page prospecting workflow, especially against budget self-serve alternatives with public pricing.

## Cite
- Key features: Chrome extension surfacing company/contact intelligence, Prospect-fit score against user-defined ICP, Filters by seniority and department, Preview phone/email/LinkedIn before spending credits, Manual contact-sourcing requests, 120M+ contacts / 21M+ direct dials / 7M companies
- Strengths: Public self-serve pricing ($0/$50mo/$480yr); manual sourcing fallback when sites lack contact details; ICP-fit score on every plan
- Gaps: 120M contacts vs 500M; no technographic intent at scale; no CRM enrichment automation; no conversation intelligence; no API or MCP
