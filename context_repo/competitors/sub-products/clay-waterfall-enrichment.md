# Clay — Waterfall Enrichment

**Competes with:** Data (Pillar 1) — direct overlap; secondary overlap with ZoomInfo Operations
**Spotlight rank:** 1

## What it does
Runs requests across multiple data providers in priority order, falling back to next provider when primary returns no match — maximizes coverage on contact info (work emails, personal emails, mobile numbers).

## Spotlight when
The article is about data coverage, enrichment cost, or the architectural debate between single-vendor vs. multi-vendor data sourcing. ZoomInfo's structural counter is verified-data rigor (95%+, 300+ human researchers) and GTM Context Graph as the reasoning layer above raw data.

## Cite
- Key features: Multi-provider waterfall on emails/mobiles, 150+ providers, wholesale rates, custom data point waterfalling, transparent attribution
- Strengths: "Use them all" architectural pitch; wholesale-rate transparency; OpenAI 30% → 80% coverage proof
- Gaps: Verification only as good as providers; per-enrichment cost scaling; no GTM Context Graph reasoning layer
- Proof quote: "We often double or triple data coverage rates at a 1/5th or less of the cost of Zoominfo." — https://www.clay.com/faq/how-can-clay-give-me-better-data-coverage-and-pricing-than-zoominfo-or-other-data-providers
