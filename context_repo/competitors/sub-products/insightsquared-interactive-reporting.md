# InsightSquared — Interactive Reporting

**Competes with:** GTM Context Graph (adjacent overlap — pipeline review and risk-detection reports applied to first-party CRM data)
**Spotlight rank:** 6

## What it does
Pipeline review and risk-detection reports that score deals on close confidence and ICP fit — funnel progression, goal-mapping, exception detection (stalled deals, single-threaded deals).

## Spotlight when
The article enumerates pipeline-review tooling and contrasts first-party-only reporting (InsightSquared) with reasoning-on-a-context-graph (ZoomInfo). Lower priority than RevOps Dashboards because surface overlaps significantly.

## Cite
- Key features: Confidence-to-Close ML model, ICP model, funnel/goal-mapping reports, exception detection (stalled/single-threaded)
- Strengths: ML pipeline review; single-threaded detection; goal-mapping for territory
- Gaps: First-party data only; no external context graph; no agent-driven insights
- Proof quote: "Confidence-to-Close machine learning model on every opportunity" — https://www.insightsquared.com/revenue-intelligence-platform/interactive-reporting/
