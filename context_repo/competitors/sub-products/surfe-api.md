# Surfe — Surfe API

**Competes with:** APIs & MCP (direct overlap on programmatic enrichment endpoints; weaker on agent/MCP layer)
**Spotlight rank:** 4

## What it does
Programmatic enrichment API that pipes Surfe's 15+ provider waterfall enrichment directly into customer sales/marketing systems, with webhooks and n8n integration.

## Spotlight when
The article maps developer/agent enrichment endpoints. Surfe's strength is "multi-vendor waterfall via one endpoint"; ZoomInfo's APIs & MCP win on agent ecosystem and enterprise governance.

## Cite
- Key features: Email + mobile enrichment endpoints, Webhooks, n8n integration, GitHub examples, Public docs
- Strengths: Multi-vendor waterfall via one API; n8n integration; ungated developer docs
- Gaps: No MCP server; smaller endpoint scope; less mature enterprise governance
- Proof quote: "Email and mobile enrichment endpoints powered by the same 15+ provider waterfall" — https://www.surfe.com/
