# Techsalerator — Techsalerator Data Marketplace

**Competes with:** Data (Pillar 1) (direct overlap on the underlying B2B + B2C + alternative data licensing layer, with global coverage as the wedge)
**Spotlight rank:** 1

## What it does
Global B2B + B2C data marketplace — Shopify-style Data Shop sells pre-packaged city/region datasets at fixed $350-$400 list prices, plus a custom enterprise licensing flow for full firmographic, contact, technographic, ESG, real estate, weather, news, and job-posting datasets via API or batch delivery.

## Spotlight when
The article highlights buyers needing global coverage (190+ countries), B2C / alternative datasets (ESG, Real Estate, KYC, End-of-Day Prices), or e-commerce data purchasing. Techsalerator is the spotlight when the buyer needs licensable raw data rather than a GTM workflow tool.

## Cite
- Key features: 320M+ businesses across 190-219 countries with 200 fields per business, 200M+ direct emails and 140M+ personal phones, Technographic dataset covering 100M+ technologies, ESG dataset (30K public companies, 450+ fields, 4K+ ETFs) plus End-of-Day Prices and Corporate Actions, Country-by-country Data Marketplace covering 200+ territories, Shopify-style Data Shop with $350-$400 per-city dataset purchases
- Strengths: Single-vendor catalog across B2B/B2C/geospatial/alternative; truly global 190+ country coverage; e-commerce per-city pricing
- Gaps: No CRM integration surfaced; no GTM workflow surfaces; custom licensing gated to RFQ
