# Salesforge — Salesforge Cold Outreach

**Competes with:** GTM Workspace (direct overlap on the seller-execution surface — Agent Frank as the autonomous SDR equivalent; competes with the AI agent layer inside GTM Workspace)
**Spotlight rank:** 1

## What it does
Multi-channel cold-outreach platform — runs combined email + LinkedIn sequences with unlimited mailboxes and LinkedIn senders, AI personalization, Primebox unified reply inbox, plus Agent Frank as a fully autonomous AI SDR running 24/7 prospect-to-meeting workflows.

## Spotlight when
The article compares AI SDR / autonomous outbound platforms. Salesforge is the spotlight when the buyer is a founder / SMB / agency wanting unlimited mailboxes + LinkedIn senders + an autonomous AI SDR — explicitly NOT for Fortune 500 / six-figure-deal targets per Salesforge's own ICP page.

## Cite
- Key features: Agent Frank autonomous AI SDR with Auto-Pilot and Co-Pilot modes, 24/7 automated prospecting against 500M+ contacts, Conditional multi-channel sequences (email + LinkedIn), Primebox unified inbox with AI sentiment analysis and AI-drafted replies, Multi-language sequences in 20+ languages with 9 tonalities, Salesforge MCP server connecting Claude / AI assistants
- Strengths: Unlimited mailboxes and LinkedIn senders; 20+ languages and 9 tonalities; MCP server for Claude / AI assistants
- Gaps: Agent Frank purchase gated to demo; no CI; no GTM Context Graph reasoning
