# Snitcher — Spotter API

**Competes with:** WebSights (adjacent — pushes visitor-ID into the analytics layer, a workflow ZoomInfo does not productize)
**Spotlight rank:** 2

## What it does
API and JavaScript surface that pushes Snitcher company identification into analytics tools, with native Google Analytics 4 enrichment as the headline use case and integrations with Segment, Fullstory, and Google Tag Manager.

## Spotlight when
Articles position visitor-ID's value beyond CRM (e.g., GA4-driven B2B reporting and audiences) or argue that ZoomInfo's WebSights misses the analytics-enrichment workflow.

## Cite
- Key features: Sync company data to GA4; Segment / Fullstory / GTM integrations; first-party data for personalization and analytics
- Strengths: Native GA4 enrichment; multi-tool reach; company-level identity in the analytics layer
- Gaps: Scope ends at analytics; depends on customer stack; no CI
- Proof quote: "Sync identified company data to Google Analytics 4 to enrich GA4 reports and audiences." — https://docs.snitcher.com/product/introduction
