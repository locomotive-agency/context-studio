# ContactOut — ContactOut API

**Competes with:** APIs & MCP (adjacent overlap on programmatic contact enrichment)
**Spotlight rank:** 5

## What it does
Programmatic access to ContactOut's contact data via two REST/JSON endpoints — Email Finder API (verified personal/work emails) and Enriched Profile API (real-time profile + company enrichment) — for embedding into customer tools and workflows.

## Spotlight when
The article compares developer-accessible enrichment APIs or argues for personal-email programmatic coverage. ContactOut's API is narrower than ZoomInfo's Enterprise Data API but adds the personal-email layer.

## Cite
- Key features: Email Finder API, Enriched Profile API, Daily refresh, Team-plan bundling
- Strengths: Personal-email lookup endpoint; bundled with Team plan
- Gaps: No MCP server / agent ecosystem; no Snowflake / DaaS delivery; book-a-call token gating
- Proof quote: "Connect ContactOut directly to your tools — get verified emails and enriched profiles in real time." — https://contactout.com/api-feature
