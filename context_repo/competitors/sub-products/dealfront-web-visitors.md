# Dealfront — Dealfront Web Visitors (Leadfeeder)

**Competes with:** WebSights (direct overlap on the website-visitor-to-company identification surface)
**Spotlight rank:** 1

## What it does
B2B website visitor identification — IP-to-company resolution that reveals which companies are visiting the website (even if they never fill out a form), with daily ML-refreshed IP database, behavioral filters, and CRM sync.

## Spotlight when
The article spotlights ZoomInfo's WebSights / website-visitor identification, especially when the angle is EU-hosted GDPR positioning, free-tier entry, or remote-worker IP accuracy.

## Cite
- Key features: IP-to-company identification including remote-worker traffic, Identifies up to 45% of companies, Real-time Slack/email alerts, 50+ behavioral and firmographic filters, Bot/ISP traffic filtering with daily ML refresh, Automatic CRM sync
- Strengths: Free-forever Lite tier (€0/mo, 100 companies/mo); EU-hosted with GDPR Legitimate-Interest framing; ~20% IP DB refresh/month
- Gaps: No proprietary first-party intent depth; no conversation intelligence; no APIs & MCP; EUR-denominated pricing skews EU
