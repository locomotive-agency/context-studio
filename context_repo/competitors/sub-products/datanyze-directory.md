# Datanyze — Datanyze Directory

**Competes with:** Data (Pillar 1) (no overlap — Datanyze is a ZoomInfo-owned subsidiary; flagged for manual review)
**Spotlight rank:** 2

## What it does
Web-based directories of companies, people, and technology market-share data — used to identify accounts by industry, country, and tech stack via web crawling and NLP on unstructured signals (job postings, press releases).

## Spotlight when
The article discusses technographics-led prospecting or SMB tech-stack discovery and needs to acknowledge ZoomInfo's owned SMB directory surface. Spotlight the NLP + tech-adds/drops detection method.

## Cite
- Key features: 23+ industry directories, per-industry company databases (e.g., 718K Software companies), Market Share reports across CDN/MA/Email Hosting/etc., tech adds/drops signals, web crawl + NLP detection
- Strengths: Public SEO-driven directories; NLP-on-unstructured-data detection; 40M+ companies' tech choices monitored
- Gaps: Captcha-blocked for bots; no intent/conversation reasoning; no orchestration or activation
- Proof quote: "Datanyze frames technographic data (tech adds/drops, NLP on job postings) as a buying-signal layer competitors can't match for SMB-priced tools" — https://www.datanyze.com/blog/technographic-data
