# S&P Capital IQ — Market Intelligence XpressAPI

**Competes with:** APIs & MCP (adjacent overlap on data delivery patterns)
**Spotlight rank:** 3

## What it does
REST/JSON APIs that deliver Market Intelligence financial and non-financial datasets on demand without requiring customers to host the underlying database — Python SDK and API Drive bulk-download included.

## Spotlight when
The article evaluates programmatic data access patterns. Capital IQ's API is the finance-data analogue to ZoomInfo's API — same data-into-CRM pattern with very different payload.

## Cite
- Key features: REST/JSON, fundamentals on 99% of global market cap, ratings on 1M+ entities, Cross Reference, Python SDK, API Drive
- Strengths: Ratings + fixed-income + cap-structure delivery, finance identifier mapping
- Gaps: Fundamentals-only payload, high cost of integration, no MCP/agent layer
- Proof quote: "REST/JSON over HTTPS; pick-and-choose dataset access" — https://www.marketplace.spglobal.com/en/solutions/api-solutions-(61953ac7-ea64-4fac-926a-feb7f846c2be)
