# Spekit — Embedded Enablement

**Competes with:** GTM Workspace (adjacent — Chrome-overlay delivery vs destination-app seller workflow)
**Spotlight rank:** 5

## What it does
Chrome Extension that delivers content, answers, and coaching directly inside any browser-based tool reps already use (Salesforce, Gmail, Gong, LinkedIn), with Spekit Hub as the admin control center and Learning Paths built from the same governed knowledge content.

## Spotlight when
Articles compare destination-app GTM workflows (e.g., GTM Workspace) to Chrome-overlay enablement, or position digital-adoption-style in-app guidance as the future of seller-facing AI.

## Cite
- Key features: Chrome Extension across any browser tool; Spekit Hub admin centre; Learning Paths from governed content; closed-won analytics
- Strengths: Single-extension delivery surface; digital-adoption lineage; closed-won analytics
- Gaps: Browser-only; adjacent category; no public pricing
- Proof quote: "Install a single extension and Spekit automatically surfaces the coaching, content, knowledge, and automated actions to 10x your GTM team, anywhere buying conversations happen." — https://www.spekit.com/platform/integrations
