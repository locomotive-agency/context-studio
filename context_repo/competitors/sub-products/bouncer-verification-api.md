# Bouncer — Bouncer Email Verification API

**Competes with:** APIs & MCP (adjacent — programmatic verification, narrower scope than ZI's data API)

**Spotlight rank:** 2

## What it does
Real-time and batch email-verification API for embedding validation into apps, sign-up flows, and bulk pipelines.

## Spotlight when
The article frames developer-facing verification, sign-up flow protection, or bulk pipeline integration. Bouncer's 180K/hour throughput and EU residency are the spotlight angles.

## Cite
- Key features: Real-time + batch (sync/async); 500K batch size; ~180K/hr throughput; rate limits 200/min batch + 1K/min real-time; AWS EU residency; multi-org / MFA / IP allow-listing
- Strengths: High batch throughput; both real-time and batch; EU residency
- Gaps: Verification-only (no enrichment); no MCP server; no B2B contact data
- Proof quote: "Real-time and batch endpoints (synchronous and asynchronous)" — https://www.usebouncer.com/email-verification-api/
