# PitchBook — Direct Data (API + Data Feed)

**Competes with:** APIs & MCP (adjacent overlap — paid data API but no MCP / agent layer)

**Spotlight rank:** 4

## What it does
Paid add-on that delivers PitchBook's private-capital-markets dataset to customer systems via REST API or bulk data feed, powering internal dashboards and analytics.

## Spotlight when
The article surveys data APIs across categories or contrasts agent-builder ecosystems. Direct Data is the spotlight on bulk delivery for analytics use cases.

## Cite
- Key features: API access, bulk data feed, powers custom dashboards/analytics
- Strengths: Bulk delivery for portfolio analytics; analyst-validated dataset; Desktop dataset accessible programmatically
- Gaps: No MCP / agent integration; no published quotas or pricing; thin developer docs surfaced
