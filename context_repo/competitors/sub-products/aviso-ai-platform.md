# Aviso AI — Aviso Platform (Revenue Operating System)

**Competes with:** GTM Context Graph (direct overlap on the underlying time-series + ML reasoning layer)
**Spotlight rank:** 3

## What it does
Underlying data and ML infrastructure that powers every Aviso product — single pane across CRMs, time-series database, AutoML, real-time streaming, and CRM writeback.

## Spotlight when
Comparing platform/architecture differentiators. Aviso Replay's 8-quarter time-series and Aviso One's multi-CRM unification are the strongest infrastructure-level angles.

## Cite
- Key features: Aviso One (multi-CRM unification), Aviso Replay (8-quarter time-series DB), Aviso Learn (AutoML), Aviso Graph (GDPR/CCPA-compliant), Aviso LiveStream, CRM Writeback, Aviso Connect (API layer), Mobile App
- Strengths: 8-quarter time-series engine; multi-CRM unification; tenant-isolated graph
- Gaps: Internal infrastructure not external dev/agent ecosystem; Connect is integration layer not MCP; no third-party B2B data
