# Salesforce Sales Cloud — Agentforce

**Competes with:** GTM Workspace (direct overlap with the AI agent layer inside GTM Workspace)
**Spotlight rank:** 1

## What it does
Salesforce's agentic AI platform — autonomous AI agents inside the CRM, Agent Builder for custom agents, pre-built agents across sales/service/marketing use cases.

## Spotlight when
The article compares AI agents or agentic AI platforms. Frame GTM Context Graph reasoning across multiple data sources as the differentiator from Agentforce's Salesforce-bound reasoning.

- Strengths: Native Salesforce context; Agent Builder customization
- Gaps: Salesforce-bound; no cross-signal reasoning layer; pricing-per-action scales unpredictably
