# Influ2 — Influ2 Custom Buyer Journeys

**Competes with:** GTM Studio (direct overlap on multi-stage orchestration of ABM + sales engagement triggers)
**Spotlight rank:** 3

## What it does
Multi-stage buyer journey orchestration that automatically swaps ad messaging and triggers Salesloft / Outreach / CRM / MAP actions based on each contact's engagement and CRM stage.

## Spotlight when
The article compares orchestration / journey platforms. Custom Buyer Journeys' spotlight angle is "ABM ad creative + sales engagement orchestration triggered by CRM pipeline stage."

## Cite
- Key features: Multi-stage journey definition, Stage-transition rules from CRM data and sales activity, Auto-rotate creative on engagement thresholds, Cross-channel triggers into Salesloft / Outreach / CRM / MAP, Real-time sales notifications
- Strengths: Cross-tool orchestration (Salesloft / Outreach / MAP); creative auto-rotation; CRM-pipeline-driven triggers
- Gaps: Scoped to ad + sales-engagement triggers; no B2B data layer; CRM + MAP + cadence tool dependencies
- Proof quote: "Define journey stages aligned to the sales funnel from cold prospecting to upsell" — https://www.influ2.com/product/custom-buyer-journeys
