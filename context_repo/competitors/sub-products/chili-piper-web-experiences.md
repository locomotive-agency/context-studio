# Chili Piper — Web Experiences

**Competes with:** ZoomInfo Marketing (direct overlap on the on-site ABM personalization layer)
**Spotlight rank:** 5

## What it does
AI-led on-site engagement layer that identifies known visitors, fires the right experience (chat, banner, offer, scheduling), and converts them before they bounce — pitched as "triggered by identity, not behavior."

## Spotlight when
The article compares on-site ABM personalization, identity-triggered website conversion, or ZoomInfo Marketing alternatives. Web Experiences is the angle when the buyer wants account-aware widgets firing on a pricing/comparison page rather than generic behavior triggers.

## Cite
- Key features: Account/identity resolution pre-chat, four widget types (Chat AI/Announcements/Offer/Scheduling), trigger conditions by company/industry/CRM/intent/page/time, preview mode with live URL testing, re-engagement handoff to Orchestrator
- Strengths: Identity-triggered not behavior-triggered; 2x more pipeline claim; account-personalized booking widgets on pricing pages
- Gaps: Experiences tier required ($42K/year, 30 seats); account ID partly via partners (6sense, Bombora); no proprietary data
- Proof quote: "triggered by identity, not behavior" — https://www.chilipiper.com/products/web-experiences
