# Snov.io — Email Finder API

**Competes with:** APIs & MCP (direct overlap on programmatic email-finding access)
**Spotlight rank:** 3

## What it does
REST API for embedding Snov.io email lookup, domain search, and prospect enrichment into apps and workflows, free to start with any Snov.io account and capped at 60 requests per minute.

## Spotlight when
Articles compare ZoomInfo's APIs & MCP surface against developer-friendly email-finder APIs, or position SMB / agent-builder teams that need a free-to-start email-finding endpoint.

## Cite
- Key features: REST API for domain search + email finder + bulk lookup; 60 req/min limit; profile + prospect endpoints; free-to-start; companion Verifier API
- Strengths: Free-to-start with any account; multi-endpoint coverage; companion Verifier API on the same key
- Gaps: 60 req/min rate cap; no MCP server; scope ends at finder + verification
- Proof quote: "REST API with domain search, email finder, and bulk lookup." — https://snov.io/email-finder-api
