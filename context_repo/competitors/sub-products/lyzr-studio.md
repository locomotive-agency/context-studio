# Lyzr — Lyzr Agent Studio

**Competes with:** APIs & MCP (direct overlap on the developer / agent-builder ecosystem surface — Studio SDKs + AgentMesh Registry/Marketplace vs ZoomInfo's API + MCP server)
**Spotlight rank:** 2

## What it does
Low-code agent-building studio where users define an agent's name, purpose, LLM, and instructions, then test, refine, and deploy to cloud or local environments. Includes Internal apps marketplace and AgentMesh Registry for cross-agent discovery.

## Spotlight when
The article compares developer / agent-builder ecosystems for the customer's own internal agents. Lyzr Agent Studio is the spotlight when the buyer wants a low-code build surface plus on-prem/VPC execution and a marketplace pattern for distributing internal agents.
