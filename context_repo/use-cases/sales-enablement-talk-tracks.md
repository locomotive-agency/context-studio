# Sales Enablement Talk Tracks

Enablement teams can request audience profiles, value proposition records, proof points, and cited transcript snippets to create talk tracks.

## Required Context

- Audience profile for the seller or buyer group.
- Product value proposition.
- Approved proof points.
- Collection snippets for recent objections and buyer language.

## Governance Rule

Transcript snippets can inform phrasing but cannot replace approved proof or claims.
