# Writing Standards

Write in short paragraphs. Lead with the problem, then name how governed context changes the workflow.

## Rules

- Use "Documents" for governed OKF records.
- Use "Collections" for searchable supporting source material.
- Use "source" when referring to an original transcript, note, or file.
- Use "context package" when describing the payload an AI workflow receives.
- Avoid "RAG" as shorthand unless the piece explicitly compares retrieval patterns.

## Preferred Phrases

- "approved Documents"
- "searchable supporting sources"
- "metadata first, full text on request"
- "governed retrieval"
