# Visual Identity Direction

The visual identity should feel like a working studio: organized, current, and focused on clarity.

## Principles

- Use clean layouts with dense but readable information.
- Show real system objects such as folders, scopes, Documents, Collections, and tool results.
- Prefer restrained contrast, crisp borders, and compact controls.
- Use screenshots that show the product doing work rather than decorative mockups.
