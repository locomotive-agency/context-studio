# Demo KPI Framework

## Product Adoption

- Number of governed Documents created.
- Percentage of Documents with scope, status, criticality, and provenance.
- Number of Collection sources indexed.

## Workflow Quality

- Percentage of test bench runs returning metadata before full text.
- Number of context packages with cited Collection passages.
- Number of missing controlled-context issues found before workflow launch.
