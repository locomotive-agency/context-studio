# Context Management Category Landscape

Context Studio competes with three broad alternatives.

## Alternatives

- Shared drives and document search: easy to adopt but weak at governance and scope.
- Prompt libraries: useful for process guidance but poor at approved factual context.
- Generic vector retrieval: strong for discovery but risky when it treats all source text as equivalent.

## Differentiation

Context Studio combines deterministic governed Documents with searchable supporting Collections. The product is strongest when teams need repeatable AI workflows that can inspect metadata, respect scope, and cite raw evidence separately.
