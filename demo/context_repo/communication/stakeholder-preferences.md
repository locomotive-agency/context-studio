# Stakeholder Communication Preferences

Executives prefer concise summaries that name the business risk, the governed record used, and any missing context. Practitioners prefer examples with exact retrieved records, Collection citations, and next actions.

## Formats

- Executive brief: three bullets and one recommendation.
- Practitioner guide: context list, source citations, and workflow notes.
- Admin review: validation errors, scope coverage, and change history.
